# ModuleNotFoundError: в Python нет модуля с именем «xlsxwriter».

Ошибка Python «ModuleNotFoundError: нет модуля с именем «xlsxwriter»» возникает, когда мы забываем установить модуль xlsxwriterперед его импортом или устанавливаем его в неправильной среде.

Чтобы устранить ошибку, установите модуль, выполнив pip install xlsxwriter команду.

Откройте терминал в корневом каталоге вашего проекта и установите модуль xlsxwriter .

оболочка

# 👇️ In a virtual environment or using Python 2

pip install xlsxwriter

# 👇️ For Python 3 (could also be pip3.10 depending on your version)

pip3 install xlsxwriter

# 👇️ If you get a permissions error

sudo pip3 install xlsxwriter

pip install xlsxwriter --user

# 👇️ If you don't have pip in your PATH environment variable

python -m pip install xlsxwriter

# 👇️ For Python 3 (could also be pip3.10 depending on your version)

python3 -m pip install xlsxwriter

# 👇️ Using py alias (Windows)

py -m pip install xlsxwriter

# 👇️ For Anaconda

conda install -c conda-forge xlsxwriter

# 👇️ For Jupyter Notebook

!pip install xlsxwriter



После установки пакета xlsxwriter попробуйте импортировать его следующим образом.

main.py

import xlsxwriter

workbook = xlsxwriter.Workbook('example.xlsx')

worksheet = workbook.add_worksheet()

worksheet.write('A1', 'First cell')

workbook.close()



# Распространенные причины возникновения ошибки

Ошибка Python возникает по нескольким причинам:

  1. Отсутствие xlsxwriterустановки пакета при запуске pip install xlsxwriter.
  2. Установка пакета в версии Python, отличной от той, которую вы используете.
  3. Установка пакета глобально, а не в вашей виртуальной среде.
  4. В вашей IDE используется неверная версия Python.
  5. Назовите свой модуль xlsxwriter.py, который будет заменять официальный модуль.
  6. Объявление переменной с именем xlsxwriter, которое будет скрывать импортированную переменную.

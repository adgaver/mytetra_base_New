Как красиво распечатать словарь на Python?

Мне нравится pprint модуль (Pretty Print), включенный в Python. Его можно использовать либо для печати объекта, либо для форматирования его хорошей строковой версии.

import pprint


# Prints the nicely formatted dictionary

pprint.pprint(dictionary)


# Sets 'pretty_dict_str' to the formatted string value

pretty_dict_str = pprint.pformat(dictionary)

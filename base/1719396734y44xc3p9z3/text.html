Как записать журналы консоли с помощью Playwright?

Playwright — это мощный инструмент для автоматизации веб-браузеров, который обеспечивает простой способ сбора журналов консоли с веб-страницы.

Вот как это можно сделать:

Питон

В Python вы можете использовать этот Page.on("console")метод для прослушивания событий журнала консоли. Вот пример:

Копировать

from playwright.sync_api import sync_playwright

def log(msg):

print(msg.text)

with sync_playwright() as p:

browser = p.chromium.launch()

page = browser.new_page()

page.on("console", log)

page.goto("https://example.com")

browser.close()



В этом примере log— это функция-обработчик, которая вызывается всякий раз, когда происходит событие консоли. Свойство msg.textсодержит сообщение журнала.

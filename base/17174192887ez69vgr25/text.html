Как добавить новый лист в существующий Excel с помощью Python

Создание файлов Excel и управление ими — обычная задача во многих проектах программирования. Иногда вам нужно добавить новый лист в существующий файл Excel. Python с помощью таких библиотек, как openpyxl , позволяет легко сделать это всего за несколько строк кода. В этом уроке мы расскажем, как добавить новый лист в существующий файл Excel с помощью Python.

from openpyxl import load_workbook
 
# Load existing workbook
filename = "sample.xlsx"
workbook = load_workbook(filename)
 
# Create a new sheet and add it to the workbook
new_sheet = workbook.create_sheet(title="New Sheet", index=1)
 
# Save the workbook
workbook.save(filename)


После выполнения приведенного выше кода вы увидите, что новый лист под названием «Новый лист» был добавлен к существующему файлу Excel в качестве второго листа. Теперь вы можете работать с этим новым листом, используя библиотеку openpyxl или любой другой инструмент редактирования Excel.

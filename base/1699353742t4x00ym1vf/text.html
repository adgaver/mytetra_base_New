Формат PDF- файла обычно используется для представления документов и обмена ими. Кроме того, он предоставляет множество преимуществ, таких как уменьшение размера в МБ и отображение текста, изображений, таблиц, аннотаций, закладок и гиперссылок без потери макета и форматирования. Разработчики Python могут учитывать эти преимущества и публиковать свои данные в формате PDF. Если вы один из таких разработчиков, эта статья поможет вам научиться создавать PDF-документ с помощью Python .

Как легко создать PDF-документ на Python

Во-первых, мы объясним, как создать PDF-документ, содержащий фрагмент текста. Ниже приведены шаги по созданию простого PDF-документа с нуля.

  1. Создайте экземпляр класса Document .
  2. Добавьте новую страницу в коллекцию страниц документа.
  3. Создайте и добавьте новый TextFragment в абзацы PDF-файла.
  4. Создайте PDF-файл с помощью метода Document.Save() .

Следующий фрагмент кода демонстрирует, как создать PDF-документ в Python.


# Initialize document object

document = aspose.pdf.Document()

# Add page

page = document.pages.add()

# Initialize textfragment object

text= "Lorem ipsum dolor sit amet, consectetur adipiscing elit, " \

"sed do eiusmod tempor incididunt ut labore et dolore magna aliqua." \

" Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris" \

" nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in " \

"reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur." \

" Excepteur sint occaecat cupidatat non proident, sunt in culpa qui " \

"officia deserunt mollit anim id est laborum."

text_fragment = aspose.pdf.text.TextFragment (text)

# Add text fragment to new page

page.paragraphs.add(text_fragment)

# Save updated PDF

document.save("output.pdf")

Применение форматирования текста в PDF-файлах с помощью Python

После создания документа давайте научимся применять форматирование текста в PDF-файлах с помощью библиотеки Python PDF. Пожалуйста, следуйте инструкциям ниже.

  • Сначала создайте новый PDF-документ, используя класс Document .
  • Во-вторых, выберите страницу, на которой вы хотите разместить текст.
  • Затем создайте объекты TextFragment и задайте для них текст и другие параметры форматирования, такие как положение, шрифт, цвет, размер и т. д.
  • После этого добавьте фрагменты текста на страницу с помощью метода Page.paragraphs.add() .
  • Наконец, вызовите метод Document.Save() , чтобы создать документ PDF.

В следующем фрагменте кода показано, как программно применить форматирование текста в PDF-файлах с помощью Python.


# Initialize document object

document = aspose.pdf.Document()

# Add page

page = document.pages.add()

# Create text fragments

text1= "Text1: Lorem ipsum dolor sit amet, consectetur adipiscing elit, " \

"sed do eiusmod tempor incididunt ut labore et dolore magna aliqua." \

" Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris" \

" nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor"

text2= "Text2: Lorem ipsum dolor sit amet, consectetur adipiscing elit, " \

"sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."

text_fragment1 = aspose.pdf.text.TextFragment (text1)

text_fragment1.position= aspose.pdf.text.Position(100, 700)

text_fragment2 = aspose.pdf.text.TextFragment (text2)

text_fragment2.position= aspose.pdf.text.Position(100, 600)

# Set text properties

text_fragment1.text_state.font_size = 12;

text_fragment1.text_state.font = aspose.pdf.text.FontRepository.find_font("TimesNewRoman");

text_fragment1.text_state.background_color=aspose.pdf.Color().aqua

text_fragment2.text_state.underline = True

text_fragment2.text_state.strike_out = True

# Add text fragments to page

page.paragraphs.add(text_fragment1)

page.paragraphs.add(text_fragment2)

# Save updated PDF

document.save("output.pdf")

Python: вставка изображений в PDF-файл

Мы научились менять форматирование текста в предыдущем разделе. Поэтому в этом разделе мы объясним, как добавлять изображения в документы PDF. Следуйте инструкциям ниже, чтобы выполнить работу.

  1. Сначала создайте новый PDF-документ, используя класс Document .
  2. Во-вторых, получите нужную страницу, куда вы хотите вставить изображение.
  3. В-третьих, добавьте изображение на страницу, используя Page.add_image(file_path, прямоугольник) , тогда как класс Rectangle используется для размещения изображения на странице.
  4. Наконец, сгенерируйте PDF-документ, используя метод Document.Save() .

В следующем примере кода показано , как добавлять изображения в PDF с помощью Python.


# Initialize document object

document = aspose.pdf.Document()

# Add page

page = document.pages.add()

# Set image coordinates

lowerLeftX = 400;

lowerLeftY = 400;

upperRightX = 150;

upperRightY = 150;

rectangle= aspose.pdf.Rectangle(lowerLeftX,lowerLeftY,upperRightX,upperRightY,True)

# Add image to page

page.add_image("aspose-logo.jpg", rectangle)

# Save updated PDF

document.save("output.pdf")

Добавьте таблицы в PDF с помощью Python

После вставки изображения давайте добавим таблицу в ваш PDF-файл. Пожалуйста, напишите код, используя шаги, указанные ниже:

  1. Создайте объект класса Document , чтобы создать новый PDF-файл.
  2. Получите страницу, на которой вы хотите создать таблицу.
  3. Создайте экземпляр класса Table .
  4. Укажите границы таблицы и ячеек с помощью класса BorderInfo .
  5. Создайте и добавьте новую строку в коллекцию Table.Rows .
  6. Добавьте ячейки в коллекцию Row.Cells .
  7. Добавьте таблицу на страницу с помощью метода Page.paragraphs.add() .
  8. Сохраните PDF-документ с помощью метода Document.Save() .

В следующем примере кода показано, как добавить таблицу в файл PDF с помощью Python.


# Initialize document object

document = aspose.pdf.Document()

# Add page

page = document.pages.add()

# Create table

table = aspose.pdf.Table()

# Define table and cell borders

table.border=aspose.pdf.BorderInfo(aspose.pdf.BorderSide.ALL,2,aspose.pdf.Color().dark_gray)

table.default_cell_border= aspose.pdf.BorderInfo(aspose.pdf.BorderSide.ALL,2,aspose.pdf.Color().black)

# Add rows in the table

for i in range(1, 11):

row = table.rows.add()

# Add table cells

row.cells.add("Column (" + str(i) + ", 1)");

row.cells.add("Column (" + str(i) + ", 2)");

row.cells.add("Column (" + str(i) + ", 3)");

# Add table to the page

page.paragraphs.add(table);

# Save updated PDF

document.save("D:\\AsposeSampleResults\\output.pdf")

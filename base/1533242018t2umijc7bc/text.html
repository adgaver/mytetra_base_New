Access. Репликация и JRO.

(обращений: 12276 с 27.06.2005)

Разделы:  Разное, Статьи, Базы

 

Описание: В этой статье рассматриваются вопросы создания и управления репликами. Здесь также описывается новый объект Jet Replication Object (JRO), предназначенный для реализации функциональных возможностей репликации и использования гибкости ее средств. Применимо для Access 2000, 2002(XP), 2003. В Access 2007, 2010, формат accdb, репликация убрана разработчиками.

Понимание временных меток

Временные метки являются важнейшими элементами в современном программировании, особенно для задач, включающих данные временных рядов или измерение времени, прошедшего между двумя событиями. В этом разделе мы рассмотрим, что такое временные метки, как они работают с Python и разницу между наивными и осведомленными временными метками.

Временная метка Unix

Временная метка Unix, также известная как время эпохи, — это количество секунд, прошедших с 1 января 1970 года, 00:00:00 UTC. Временная метка Unix широко используется как стандартизированный метод представления времени на основе стандарта временных меток POSIX . Некоторые из преимуществ использования временных меток Unix:

  • Переносимость: работает с различными операционными системами и языками программирования.
  • Простое сравнение: сравнение временных меток в секундах упрощает расчет длительностей и временных интервалов.
  • Компактное представление: целые числа проще хранить, передавать и обрабатывать, чем сложные объекты даты и времени.

Объект Python datetime

В Python модуль datetime предоставляет классы для работы с датами и временем без усилий. Наиболее часто используемые классы:

  • date: Обозначает дату (год, месяц и день) в григорианском календаре.
  • time: Представляет время (часы, минуты, секунды и микросекунды) в 24-часовом формате.
  • datetime: Объединяет функции даты и времени в один объект.

Для работы с временными метками можно использовать datetimeметоды класса, такие как fromtimestamp()и timestamp(), для преобразования временных меток Unix в datetimeобъекты и наоборот.

Это импортируемые библиотеки, которые вам понадобятся сегодня, поэтому обязательно прикрепите их в верхней части вашего скрипта/блокнота:

import pytz

from datetime import datetime, date, time, timedelta, timezone

Наивные и осознанные временные метки

Модуль Python datetimeразличает два типа временных меток: наивные и осведомленные .

  • Наивные временные метки не включают никакой информации о часовом поясе, то есть они регистрируют только местное время без учета летнего времени или других корректировок. В результате наивные временные метки могут быть неоднозначными, поскольку они не указывают на конкретное место или контекст, в котором было записано время.
  • Знающие временные метки включают часовой пояс как часть объекта, предлагая более точное представление фактического времени в определенном месте. Добавляя часовой пояс, вы можете работать с точными измерениями времени, которые всегда учитывают любые контекстные изменения в хронометрировании.

Форматирование даты и времени в Python

Функция strftime()позволяет форматировать datetimeобъект в строку с определенным форматом. Вы можете использовать коды формата для представления компонентов даты и времени. Вот некоторые часто используемые коды формата:

  • %Y- Год с веком
  • %y- Год без века
  • %m- Месяц в виде десятичного числа с добавлением нулей
  • %d- День месяца в виде десятичного числа с добавлением нулей
  • %H- Час (24-часовой формат) в виде десятичного числа с добавлением нулей
  • %M- Минута как десятичное число с добавлением нулей
  • %S- Секунда как десятичное число с добавлением нулей

Вот пример форматирования datetimeобъекта с помощью strftime():

dt = datetime(2023, 6, 21, 17, 35, 22)

formatted_dt = dt.strftime("%Y-%m-%d %H:%M:%S")

print(f"Formatted datetime: {formatted_dt}")

Из даты и времени в метку времени Unix

Преобразование объекта datetime в метку времени Unix — это простой процесс с использованием .timestamp()метода. Обработка високосных секунд

Високосные секунды — это случайные корректировки на одну секунду для учета замедления вращения Земли. Они могут происходить в различных временных метках в пределах одной секунды, приводя к одному и тому же значению временной метки POSIX.

Библиотека Python timeобрабатывает високосные секунды в соответствии с внутренней конфигурацией системы. Однако в ней нет прямого метода добавления високосных секунд при обработке времени. Чтобы обеспечить согласованность и избежать проблем с високосными секундами, вы можете использовать библиотеку astropy.time.

Но сначала вам нужно его установить:

pip install astropy



А теперь перейдем к примеру использования:

from astropy.time import Time

t = Time("2023-06-21T00:00:00", scale="utc", format="isot")

t += 1

print(f"Time after one second (including leap second): {t.isot}")

Работа с временными метками с более высокой точностью

Встроенный пакет Python datetime обычно предоставляет временные метки с точностью до микросекунд. Однако вы можете столкнуться с временными метками с более высокой точностью, например, наносекундами. Их можно проанализировать с помощью библиотеки numpy:

import numpy as np

timestamp_str = "2023-06-21T08:17:14.078685237Z"

dt = datetime.strptime(timestamp_str[:-9], "%Y-%m-%dT%H:%M:%S.%f")

timestamp = np.datetime64(dt)

print(f"Parsed high-precision timestamp: {timestamp}")

import pandas as pd

ts = pd.Timestamp(timestamp_str)

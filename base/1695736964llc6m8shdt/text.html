Как определить значение по умолчанию, если пустой пользовательский ввод в Python?


Питон 3:

inp = int(input("Enter the inputs : ") or "42")



Питон 2:

inp = int(raw_input("Enter the inputs : ") or "42")



Как это работает?

Если ничего не было введено, input/ raw_inputвозвращает пустую строку. Пустая строка в Python — это False, bool("") -> False. Оператор orвозвращает первое истинное значение, которым в данном случае является "42".

Это не сложная проверка ввода, поскольку пользователь может ввести что угодно, например десять символов пробела, которые тогда будут True.

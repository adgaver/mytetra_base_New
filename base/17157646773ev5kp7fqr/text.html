Как преобразовать строку в DateTime?

В Python пользователи иногда могут вводить дату в виде строки, из-за чего мы не можем выполнять с ней операции с датой и временем. Такие операции, как разница во времени, сравнение дат, работа с часовыми поясами и т.д., не могут быть выполнены со строковыми датами.

Давайте рассмотрим процесс преобразования строк в даты с помощью двух методов.

Input: Dec 4 2018 10:07 AM
Output: 2018-12-04 10:07:00
Explanation: In this, we are converting the string to Date Time.


1. Преобразуйте строку в Datetime с помощью Datetime.Strptime()

Функция, и она используется для преобразования даты во время. Эта функция изменяет заданную строку Datetime в желаемый формат.

Пример: Программа определяет функцию, которая преобразует дату и время, представленные в виде строки, в объект datetime , используя желаемый формат. Преобразование выполняется с использованием метода datetime.strptime().

  • Python3


import datetime

  

# Function to convert string to datetime

def convert(date_time):

    format = '%b %d %Y %I:%M%p'

    datetime_str = datetime.datetime.strptime(date_time, format)

  

    return datetime_str

    

date_time = 'Dec 4 2018 10:07AM'

print(convert(date_time))

print(type(convert(date_time)))

Вывод:

2018-12-04 10:07:00
<class 'datetime.datetime'>)


2. Преобразуйте строку в Datetime с помощью модуля Dateutil

В Dateutil это модуль сторонних производителей. Модуль Dateutil поддерживает синтаксический анализ дат в любом строковом формате. Этот модуль предоставляет внутреннюю информацию о текущих часовых поясах мира. Parse() может преобразовать строку Python в формат даты и времени. Единственным используемым параметром является строка.

Пример: Программа импортирует модуль синтаксического анализа библиотеки dateutil для преобразования строки в datetime , который имеет функцию для синтаксического анализа строк даты и времени. Функция Parser.parse() автоматически распознает формат, который преобразует строку “Jun 23 2022 07:31 PM” в объект DateTime.

  • Python3


from dateutil import parser

  

DT = parser.parse("Jun 23 2022 07:31PM")

  

print(DT)

print(type(DT))

Вывод:

2022-06-23 19:31:00
<class 'datetime.datetime'>

Как преобразовать Datetime в строку?

Вы также можете преобразовать DateTime в строку для различных целей, таких как создание отчетов, пользовательское форматирование или форматирование для отображения. Давайте рассмотрим различные методы преобразования datetime в строку.

Input: 2018-12-04 10:07:00
Output: Dec 4 2018 10:07:00AM
Explanation: In this, we are converting the datetime format to the string


1. Преобразуйте Datetime в строку using Time.strftime()

ФункцияPython, и она позволяет создавать строковое представление на основе указанной строки формата. Мы используем этот метод для преобразования datetime в строку в Python.

Другая аналогичная функция доступна в модуле Python time, который преобразует объект tuple или struct_time в строку, указанную аргументом format .

Пример: Программа преобразует Datetime в строку и импортирует модуль Python time, который предлагает функции для обработки действий, связанных со временем. Функция convert() принимает кортеж datetime_str в качестве входных данных, преобразует его в временную метку с помощью time.mktime(), а затем форматирует временную метку в строковое представление, используя формат “%b%d%Y% r”, который был предоставлен.

  • Python3


import time

  

# Function to convert string to datetime

def convert(datetime_str):

    datetime_str = time.mktime(datetime_str)

    format = "%b %d %Y %r"  

    dateTime = time.strftime(format, time.gmtime(datetime_str))

    return dateTime

  

date_time = (2018, 12, 4, 10, 7, 00, 1, 48, 0)

print(convert(date_time))

print(type(convert(date_time)))

Вывод:

Dec 04 2018 04:37:00 AM
<class 'str'>


Временная сложность: O(1)
Вспомогательный пробел: O(1)

2. Преобразуйте Datetime в строку с помощью Datetime

Используйте модуль datetime для преобразования datetime в строку, используя указанные дату и время, и создайте объект datetime. Создайте строку формата, которая описывает требования к форматированию для объекта datetime.

Чтобы создать строку в желаемом формате, используйте метод strftime() объекта DateTime. Строка формата является аргументом функции strftime().

Пример: Программа импортирует модуль datetime, который предоставляет программистам Python доступ к классам для работы с датами и временем. Предоставленные значения даты и времени, 14 марта 2022 года, в 15:30:00, используются для создания объекта datetime. Объект datetime преобразуется в строковое представление на основе строки формата с использованием метода strftime().

  • Python3


import datetime

  

datetime_object = datetime.datetime(2022, 3, 14, 15, 30, 0)

  

format_string = '%Y-%m-%d %H:%M:%S'

  

# Convert the datetime object to a string in the specified format

date_string = datetime_object.strftime(format_string)

  

print(date_string)

print(type(date_string))

Вывод:

2022-03-14 15:30:00
<class 'str'>


Временная сложность: O(1)
Вспомогательный пробел: O(1)

Действия

Введение

Драматург может взаимодействовать с элементами HTML-ввода, такими как ввод текста, флажки, переключатели, параметры выбора, щелчки мыши, ввод символов, клавиши и сочетания клавиш, а также загружать файлы и элементы фокуса.

Ввод текста

Использование locator.fill() — самый простой способ заполнить поля формы. Он фокусирует элемент и вызывает inputсобытие с введенным текстом. Это работает для <input>элементов <textarea>и [contenteditable].

// Text input
await page.getByRole('textbox').fill('Peter');

// Date input
await page.getByLabel('Birth date').fill('2020-02-02');

// Time input
await page.getByLabel('Appointment time').fill('13:15');

// Local datetime input
await page.getByLabel('Local time').fill('2020-03-02T05:15');


Флажки и переключатели

Использование locator.setChecked() — это самый простой способ установить и снять флажок с флажка или переключателя. Этот метод можно использовать с input[type=checkbox]элементами input[type=radio]и [role=checkbox].

// Check the checkbox
await page.getByLabel('I agree to the terms above').check();

// Assert the checked state
expect(page.getByLabel('Subscribe to newsletter')).toBeChecked();

// Select the radio button
await page.getByLabel('XL').check();


Выберите параметры

Выбирает один или несколько параметров в <select>элементе с помощью locator.selectOption() . Вы можете указать опцию value, или labelвыбрать. Можно выбрать несколько вариантов.

// Single selection matching the value or label
await page.getByLabel('Choose a color').selectOption('blue');

// Single selection matching the label
await page.getByLabel('Choose a color').selectOption({ label: 'Blue' });

// Multiple selected items
await page.getByLabel('Choose multiple colors').selectOption(['red', 'green', 'blue']);


Щелчок мышью

Выполняет простой человеческий щелчок.

// Generic click
await page.getByRole('button').click();

// Double click
await page.getByText('Item').dblclick();

// Right click
await page.getByText('Item').click({ button: 'right' });

// Shift + click
await page.getByText('Item').click({ modifiers: ['Shift'] });

// Hover over element
await page.getByText('Item').hover();

// Click the top left corner
await page.getByText('Item').click({ position: { x: 0, y: 0 } });


Под капотом этот и другие методы, связанные с указателями:

  • дождаться, пока элемент с данным селектором окажется в DOM
  • дождитесь, пока оно отобразится, т.е. не будет пустым, нет display:none, нетvisibility:hidden
  • подождите, пока он перестанет двигаться, например, пока не завершится переход css
  • прокрутить элемент в поле зрения
  • дождаться, пока он получит события указателя в точке действия, например, ждет, пока элемент не перестанет перекрываться другими элементами
  • повторите попытку, если элемент был отсоединен во время любой из вышеуказанных проверок

Принудительное нажатие

Иногда приложения используют нетривиальную логику: при наведении на элемент накладывается на него другой элемент, который перехватывает щелчок. Такое поведение неотличимо от ошибки, при которой элемент закрывается, а щелчок отправляется в другое место. Если вы знаете, что это происходит, вы можете обойти проверки работоспособности и принудительно нажать:

await page.getByRole('button').click({ force: true });


Программный клик

Если вы не заинтересованы в тестировании своего приложения в реальных условиях и хотите имитировать щелчок любыми возможными способами, вы можете инициировать это HTMLElement.click()поведение, просто отправив событие щелчка по элементу с помощью locator.dispatchEvent() :

await page.getByRole('button').dispatchEvent('click');


Введите символы

ОСТОРОЖНОСТЬ

В большинстве случаев вам следует вводить текст с помощью locator.fill() . См. раздел «Ввод текста» выше. Вам нужно вводить символы только в том случае, если на странице предусмотрена специальная обработка клавиатуры.

Введите в поле символ за символом, как если бы это был пользователь с настоящей клавиатурой с помощью locator.pressSequentially() .

// Press keys one by one
await page.locator('#area').pressSequentially('Hello World!');


Этот метод будет генерировать все необходимые события клавиатуры со всеми событиями keydown, keyup, keypressна месте. Вы даже можете указать необязательное значение delayмежду нажатиями клавиш, чтобы имитировать поведение реального пользователя.

Клавиши и ярлыки

// Hit Enter
await page.getByText('Submit').press('Enter');

// Dispatch Control+Right
await page.getByRole('textbox').press('Control+ArrowRight');

// Press $ sign on keyboard
await page.getByRole('textbox').press('$');


Метод locator.press() фокусирует выбранный элемент и производит одно нажатие клавиши. Он принимает имена логических клавиш, которые создаются в свойстве KeyboardEvent.key событий клавиатуры:

Backquote, Minus, Equal, Backslash, Backspace, Tab, Delete, Escape,
ArrowDown, End, Enter, Home, Insert, PageDown, PageUp, ArrowRight,
ArrowUp, F1 - F12, Digit0 - Digit9, KeyA - KeyZ, etc.


  • Альтернативно вы можете указать один символ, который хотите создать, например "a"или "#".
  • Также поддерживаются следующие сочетания клавиш для модификации: Shift, Control, Alt, Meta.

Простая версия создает один символ. Этот символ чувствителен к регистру, поэтому "a"будет "A"давать разные результаты.

// <input id=name>
await page.locator('#name').press('Shift+A');

// <input id=name>
await page.locator('#name').press('Shift+ArrowLeft');


Ярлыки, такие как "Control+o"или, "Control+Shift+T"также поддерживаются. Если указано с модификатором, модификатор нажимается и удерживается, пока нажимается последующая клавиша.

Обратите внимание, что вам все равно нужно указать прописную букву, Aчтобы Shift-Aсоздать заглавную букву. Shift-aвыдает строчные буквы, как если бы у вас было CapsLockпереключение.

Загрузить файлы

Выбрать входные файлы для загрузки можно с помощью метода locator.setInputFiles() . Он ожидает, что первый аргумент укажет на входной элемент типа "file". В массиве можно передавать несколько файлов. Если некоторые пути к файлам являются относительными, они разрешаются относительно текущего рабочего каталога. Пустой массив очищает выбранные файлы.

// Select one file
await page.getByLabel('Upload file').setInputFiles(path.join(__dirname, 'myfile.pdf'));

// Select multiple files
await page.getByLabel('Upload files').setInputFiles([
 path.join(__dirname, 'file1.txt'),
 path.join(__dirname, 'file2.txt'),
]);

// Remove all the selected files
await page.getByLabel('Upload file').setInputFiles([]);

// Upload buffer from memory
await page.getByLabel('Upload file').setInputFiles({
 name: 'file.txt',
 mimeType: 'text/plain',
 buffer: Buffer.from('this is test')
});


Если у вас нет элемента ввода (он создается динамически), вы можете обработать событие page.on('filechooser') или использовать соответствующий метод ожидания для вашего действия:

// Start waiting for file chooser before clicking. Note no await.
const fileChooserPromise = page.waitForEvent('filechooser');
await page.getByLabel('Upload file').click();
const fileChooser = await fileChooserPromise;
await fileChooser.setFiles(path.join(__dirname, 'myfile.pdf'));


Фокусный элемент

Для динамических страниц, которые обрабатывают события фокуса, вы можете сфокусировать данный элемент с помощью locator.focus() .

await page.getByLabel('Password').focus();


Перетащите

Вы можете выполнить операцию перетаскивания с помощью locator.dragTo() . Этот метод будет:

  • Наведите курсор на элемент, который будет перетаскиваться.
  • Нажмите левую кнопку мыши.
  • Подведите указатель мыши к элементу, который получит каплю.
  • Отпустите левую кнопку мыши.

await page.locator('#item-to-be-dragged').dragTo(page.locator('#item-to-drop-at'));


Перетаскивание вручную

Если вам нужен точный контроль над операцией перетаскивания, используйте методы более низкого уровня, такие как locator.hover() , mouse.down() , mouse.move() и mouse.up() .

await page.locator('#item-to-be-dragged').hover();
await page.mouse.down();
await page.locator('#item-to-drop-at').hover();
await page.mouse.up();


ПРИМЕЧАНИЕ

Если ваша страница зависит от dragoverотправки события, вам потребуется как минимум два движения мыши, чтобы вызвать его во всех браузерах. Чтобы надежно выполнить второе движение мыши, дважды повторите команду mouse.move() или locator.hover() . Последовательность операций будет такой: наведите курсор на перетаскиваемый элемент, указатель мыши вниз, наведите курсор на перетаскиваемый элемент, наведите курсор на перетаскиваемый элемент второй раз, указатель мыши вверх.

Как обрабатывать действия мыши с помощью Playwright?

Playwright — мощный инструмент для автоматизации действий браузера, таких как щелчки мыши, ввод с клавиатуры и даже сложные движения мыши. Он доступен для JavaScript, Python и других языков. В этом посте мы покажем, как выполнять действия мышью с помощью Playwright в JavaScript и Python.

JavaScript

В JavaScript вы можете использовать этот page.mouseкласс для перемещения курсора и выполнения щелчков мыши. Вот пример:

Копировать

const { chromium } = require('playwright'); // Or 'firefox' or 'webkit'.

(async () => {

const browser = await chromium.launch();

const page = await browser.newPage();

await page.goto('http://example.com');

// Move the mouse to coordinates (100, 100) on the page.

await page.mouse.move(100, 100);

// Perform a left mouse button click.

await page.mouse.click(100, 100);

// Perform a right mouse button click.

await page.mouse.click(100, 100, { button: 'right' });

// Perform a double click.

await page.mouse.dblclick(100, 100);

await browser.close();

})();



Питон

В Python вы можете использовать page.mouseкласс для движений мыши и щелчков мышью. Вот пример:

Копировать

from playwright.sync_api import sync_playwright

with sync_playwright() as p:

browser = p.chromium.launch()

page = browser.new_page()

page.goto('http://example.com')

# Move the mouse to coordinates (100, 100) on the page.

page.mouse.move(100, 100)

# Perform a left mouse button click.

page.mouse.click(100, 100)

# Perform a right mouse button click.

page.mouse.click(100, 100, button='right')

# Perform a double click.

page.mouse.dblclick(100, 100)

browser.close()



Помните, что координаты, (100, 100)используемые в этих примерах, предназначены только для демонстрации. Вам необходимо заменить их фактическими координатами на вашей веб-странице.

Дополнительное примечание:

Если вы хотите выполнять сложные действия с мышью, такие как перетаскивание, вы можете сделать это, объединив методы move, down, и upкласса mouse.

Например, следующий код JavaScript перетащит элемент из (startX, startY)в (endX, endY):

Копировать

await page.mouse.move(startX, startY);

await page.mouse.down();

await page.mouse.move(endX, endY);

await page.mouse.up();



То же самое можно добиться в Python следующим образом:

Копировать

page.mouse.move(startX, startY)

page.mouse.down()

page.mouse.move(endX, endY)

page.mouse.up()



В этих примерах замените startX, startY, endXи endYсвоими фактическими координатами.

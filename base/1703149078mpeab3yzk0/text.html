НАПИСАНИЕ ТЕКСТОВЫХ ФАЙЛОВ В VBA

Подход №1: Функция Write (строки в кавычках)

Запись строк данных в текстовые файлы с кавычками:

Dim fileName As String, textData As String, textRow As String, fileNo As Integer

fileName = "C:\test.txt"

fileNo = FreeFile 'Get first free file number 

textData ="Hello World!"

 

Open fileName For Output As #fileNo  'Open file for overwriting! Replace Output with Append to append

Write #fileNo, textData

Close #fileNo

Этот подход приведет к следующему результату:


1

"Hello, World!"

Обратите внимание на кавычки «» . Запись дополнительных строк приведет к добавлению дополнительных строк, заключенных в двойные кавычки, что может быть неудовлетворительным подходом. Посмотрите на функцию «Печать», чтобы избежать этих кавычек.

Чтобы добавить к текстовому файлу вместо перезаписи, замените «Для вывода» на «Для добавления» в имени открытого файла. Для вывода как #fileNo.

На этот раз в файле нет кавычек. В большинстве случаев вам захочется использовать функцию печати вместо функции записи.

Dim fileName As String, fileNo As Integer, testVar As Integer

fileName = "C:\test.bin"

testVar = 4

fileNo = FreeFile

Open fileName For Binary Lock Read Write As #fileNo

Put #fileNo, , testVar

Close #fileNo

В двоичных файлах вы часто будете использовать объекты, которые не имеют фиксированной длины в байтах, например целые числа . Например, вы хотите сохранить строки. В таких случаях используйте тип данных объекта VBA Type .

Ниже приведен простой пример использования типа данных Type для сохранения объекта с целым числом и строкой.

Type TestType

    intVar As Integer

    strVar As String

End Type

 

Sub WriteBinary()

  Dim fileName As String, fileNo As Integer, testVar As TestType

  fileName = "C:\test.bin"

  testVar.intVar = 4

  testVar.strVar = "Hello!"

  fileNo = FreeFile

  Open fileName For Binary Lock Read Write As #fileNo

  Put #fileNo, , testVar

  Close #fileNo

End Sub

НАПИСАНИЕ CSV-ФАЙЛОВ В VBA

Если вы хотите сохранить рабочий лист в формате CSV, вы всегда можете воспользоваться разделом «Текстовый файл» . Однако это неэффективный подход. Если вы хотите сохранить текущий рабочий лист в виде файла, разделенного точкой с запятой ( ';' ), вы можете использовать функцию Workbook.SaveAs , как показано ниже.


1

2

3

4

PathName = "C:\test.csv"

'Assuming you want to export your current Worksheet

ActiveSheet.Move 'Move the Worksheet to a new Workbook

ActiveWorkbook.SaveAs Filename:=PathName, FileFormat:=xlCSV, CreateBackup:=False

Если вы хотите вместо этого скопировать рабочий лист, просто замените Move на Copy :


1

2

3

4

PathName = "C:\test.csv"

'Assuming you want to export your current Worksheet

ActiveSheet.Copy 'Copy the Worksheet to a new Workbook

ActiveWorkbook.SaveAs Filename:=PathName, FileFormat:=xlCSV, CreateBackup:=False

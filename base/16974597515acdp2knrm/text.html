datetime и классы datetime.timedelta — ваши друзья.

  1. найди сегодня.
  2. используйте это, чтобы найти первый день этого месяца.
  3. используйте timedelta для резервного копирования одного дня, до последнего дня предыдущего месяца.
  4. напечатайте строку ГГГГММ, которую вы ищете.

Так:

import datetime

today = datetime.date.today()

first = today.replace(day=1)

last_month = first - datetime.timedelta(days=1)

print(last_month.strftime("%Y%m"))

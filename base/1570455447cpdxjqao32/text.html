Как перейти на «Skip Ahead» сборки Windows 10

2019-01-30 | Автор Comss.one | комментарии
Опция «Skip Ahead» (Перейти к следующей версии Windows) позволяет участникам программы Windows Insider переходить непосредственно к Pre-release сборкам следующего крупного обновления Windows 10

Microsoft приступила к завершающей стадии разработки обновления Windows 10 (версия 1903) (19H1), которое может получить название Windows 10 May 2019 Update. Это означает, что тестовые сборки станут более стабильными и менее “авантюрными”.

В результате они могут представлять меньше интереса для некоторых участников программы предварительной оценки, которые привыкли к трудностям и желают принимать наиболее полное участие в разработке Windows 10. Как раз для таких участников Microsoft предлагает опцию Skip Ahead, которая позволяет перейти к тестированию следующего выпуска Windows 10.

  • Настройка Skip Ahead через Параметры Windows 10
  • Настройка Skip Ahead через реестр

Настройка Skip Ahead через Параметры Windows 10

Примечание

Microsoft закрыла регистрацию в Skip Ahead через Параметры. Используйте Настройка Skip Ahead через реестр

Опция Skip Ahead (Перейти) позволяет участникам программы Windows Insider переходить непосредственно к сборкам Pre-release следующего крупного обновления Windows 10 20H1, которое планируется к запуску весной 2020 года.

Перейдите в меню Параметры > Обновление и безопасность > Программа предварительной оценки Windows и для опции Укажите параметры участника программы предварительной оценки выберите тип предварительных сборок – Перейти (Skip Ahead).

Таким образом, инсайдеры, которую выберут опцию Skip Ahead, начнут получать новые сборки раньше всех. Естественно, нужно иметь в виду, что сборки скорее всего будут достаточно “сырыми” с большим количеством ошибок и частичной потерей функциональности.

Обратите внимание, что опция Skip Ahead будет доступна временно. Количество инсайдеров со Skip Ahead может быть ограничено, потому что Microsoft все еще нуждается в тестировании весеннего обновления.

Настройка Skip Ahead через реестр

Перейти (Skip Ahead) на ветку RS_PRERELEASE

Для переходе на тестирование ветки RS_PRERELEASE (20H1) сделайте изменения в реестре или с помощью блокнота создайте и запустите файл .reg со следующим содержимым:

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost\UI\Selection]

"UIContentType"="Skip"

"UIRing"="WIF"


[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost\Applicability]

"ContentType"="Skip"

"Ring"="WIF"



Как прекратить получать сборки Insider Preview

Чтобы прекратить получать новые тестовые сборки на ваш ПК, перейдите в меню Параметры > Обновление и Безопасность > Программа предварительной оценки Windows и включите «Прекратить получение предварительных сборок».

Восстановление устройства

Если вы прекратите получать сборки Insider Preview на канале «Skip Ahead» во время цикла разработки, вам нужно будет выполнить восстановление ПК к текущей производственной сборке – для этого воспользуйтесь Media Creation Tool для установки стабильной версии или следующими инструкциями для загрузки и установки сборок Insider Preview, доступных на каналах Ранний или Поздний доступ.

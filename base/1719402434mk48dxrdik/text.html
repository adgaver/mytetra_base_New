Обработка нескольких вкладок в Playwright Python

Опубликовано arilio666​Ср, 17.05.2023 - 13:57

Чтобы работать со многими вкладками в Playwright с Python, используйте свойство Page.contexts для получения списка контекстов браузера, а затем перебирайте каждый контекст для доступа к отдельным страницам.

Пример

from playwright.sync_api import sync_playwright

def handle_multiple_tabs():

with sync_playwright() as playwright:

browser = playwright.chromium.launch(headless=False)

context = browser.new_context()

page1 = context.new_page()

page1.goto('https://www.programsbuzz.com/user/login')

print("Page 1 URL:", page1.url)

page2 = context.new_page()

page2.goto('https://www.programsbuzz.com')

print("Page 2 URL:", page2.url)

context.pages[0].bring_to_front()

print("Switched to Page 1")

page1.wait_for_load_state()

page1.locator('#edit-name').type('Monkey D Luffy')

context.pages[1].bring_to_front()

print("Switched to Page 2")

page2.locator('.fas.fa-search').click()

browser.close()

handle_multiple_tabs()


  • Мы используем playwright.chromium.launch() для запуска браузера и браузер.new_context() для создания нового контекста браузера. 
  • Затем мы используем context.new_page() для создания двух вкладок в контексте и Page.goto() для перехода на отдельные веб-сайты.
  • Для перехода между вкладками мы используем список context.pages, который содержит все открытые страницы контекста. 
  • Доступ к отдельным страницам в списке можно получить по их индексу. Context.pages[0] представляет первую вкладку, тогда как context.pages[1] описывает вторую.
  • Ссылаясь на различные объекты страницы (страница 1, страница 2 и т. д.) и используя API-интерфейсы Playwright, вы можете выполнять операции на каждой странице, например щелкать элементы, заполнять формы или извлекать данные.

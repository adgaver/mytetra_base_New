Как преобразовать datetime в date в Python

Последнее обновление : 22 августа 2022 г.

Улучшить

В этой статье мы увидим, как преобразовать DateTime в date в Python. Для этого мы будем использовать метод strptime() и модуль Pandas. Этот метод используется для создания объекта DateTime из строки. Затем мы извлекем дату из объекта DateTime с помощью функции date() и dt.date из Pandas в Python.

Метод 1: Преобразуйте DateTime в date в Python с помощью DateTime

Классы для работы с датой и временем предоставляются модулем Python Datetime. Эти классы предоставляют многочисленные возможности для работы с датами, временем и временными интервалами. Python обрабатывает date и DateTime как объекты, поэтому, когда вы работаете с ними, вы действительно работаете с объектами, а не со строками или временными метками.

Синтаксис strptime()

Синтаксис: datetime.strptime()

Параметры:

аргумент: Это может быть integer, float, tuple, Series, Dataframe для преобразования в datetime в качестве типа данных

формат: Это будет str, но по умолчанию - None. strftime для анализа времени, например “% d /% m /% Y", обратите внимание, что “%f” будет анализироваться с точностью до наносекунд.

Пример 1: Преобразование DateTime в дату

В этом примере мы создали datetime_str, который имеет значение “24AUG2001101010”, а его формат - “%d%b%Y%H%M%S”.

  • Python3


# import important module

import datetime

from datetime import datetime

 

# Create datetime string

datetime_str = "24AUG2001101010"

print("datetime string : {}".format(datetime_str))

 

# call datetime.strptime to convert

# it into datetime datatype

datetime_obj = datetime.strptime(datetime_str,

                                 "%d%b%Y%H%M%S")

 

# It will print the datetime object

print(datetime_obj)

 

# extract the time from datetime_obj

date = datetime_obj.date()

print(date)

Вывод: 

datetime string : 24AUG2001101010

2001-08-24 10:10:10

2001-08-24

Метод 2: Преобразуйте DateTime в date в Python с помощью Pandas

Pandas предоставляет другой набор инструментов, с помощью которых мы можем выполнять все необходимые задачи с данными о дате и времени. Давайте попробуем разобраться на примерах, рассмотренных ниже.

Пример:

Значение даты и значение DateTime отображаются в выходных данных с помощью команды print . Значения DateTime сначала добавляются в столбец фрейма данных Pandas. Затем значение DateTime преобразуется в значение даты с помощью функции dt.date().

  • Python3


import pandas as pd

df = pd.DataFrame({'time': ['2022-7-16 11:05:00',

                               '2025-7-18 12:00:30']})

print(df)

df['time'] = pd.to_datetime(df['time']).dt.date

print(df)

Вывод: 

time

0 2022-7-16 11:05:00

1 2025-7-18 12:00:30


time

0 2022-07-16

1 2025-07-18

Получить список таблиц в БД Access

Опубликовано Декабрь 8, 2015 автором Памирыч

Get table names (list all tables) in a DB MS Access / VB .NET

        Dim conn As New OleDbConnection("Provider=Microsoft.Jet.OleDb.4.0;Data Source=C:\1.mdb")

 

        conn.Open()

 

        ComboBox1.Items.Clear()

        For Each dataRow As DataRow In conn.GetSchema("Tables").Rows

            If dataRow.Item("TABLE_TYPE").ToString = "TABLE" Then

                ComboBox1.Items.Add(dataRow.Item("TABLE_NAME").ToString)

            End If

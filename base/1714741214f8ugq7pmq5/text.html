VBScript: Открытие файла программой по умолчанию


Использование оболочки Shell.Application

При использовании скриптов на VBScript может потребоваться выполнить открытие какого-либо файла, при помощи назначенного этому типу файлов приложению по умолчанию. Например, файл с расширнием rtf может открыть WordPad, Microsoft Word, LibreOffice Writer и множество других редакторов - и именно от настроек в системе зависит, при помощи какого приложения он будет открыт.

В VBScript это делается при помощи оболочки Shell.Application. Выглядит код с её использованием следующим образом:

Set objShell = CreateObject("Shell.Application")

objShell.Open "C:\temp\test.rtf"


При запуске скрипта с таким содержимым, будет открыт файл test.rtf, который находится в директории C:\temp. Для открытия файла, будет использовано назначенное данному типу файлов приложение по умолчанию.

Как работает Page.waitForLoadState в Playwright?

Page.waitForLoadState в Playwright ожидает, пока страница загрузится до определенного состояния. Существует три основных состояния загрузки страницы:

— "load" — когда страница полностью загружена, включая все ресурсы (картинки, стили, скрипты и т. д.);
— "domcontentloaded" — когда браузер построил DOM (Document Object Model) страницы, но могут быть не загружены все ресурсы;
— "networkidle" — когда браузер закончил загрузку ресурсов и не выполняет активных сетевых запросов в течение определенного периода времени.

Page.waitForLoadState позволяет ожидать любое из этих состояний, передавая его в качестве аргумента.

Пример:


const { chromium } = require('playwright');

 

(async () => {

  const browser = await chromium.launch();

  const context = await browser.newContext();

  const page = await context.newPage();

 

  await page.goto('https://example.com');

  await page.waitForLoadState('domcontentloaded'); // ожидание загрузки DOM

 

  // выполнение действий после загрузки DOM

 

  await browser.close();

})();


В этом примере мы используем Page.waitForLoadState для ожидания загрузки DOM страницы до выполнения действий далее в скрипте.

Date.ToText

Обновление

23 декабря 2023 г.

Рик де Гроот

Date.ToText — это функция Power Query M, которая возвращает текстовое представление даты с дополнительными параметрами форматирования и региональных параметров.

Совместим с :Служба Power BI Рабочий стол Power BI Excel Microsoft 365

Синтаксис

Date.ToText(

date as nullable date,

optional options as any,

optional culture as nullable text,

) as nullable text


Копировать

Аргумент

Атрибут

Описание

Дата

Значение даты, которое необходимо преобразовать в текст.

Параметры

Необязательный

Это необязательный аргумент, в котором вы можете указать параметр записи с дополнительными инструкциями. Поле Formatиспользуется для указания выходного формата преобразования. Вы также можете указать Cultureполе, чтобы гарантировать, что преобразование поддерживает локальные форматы. Например: [Format="d MMM гггг", Culture="nl-NL"]. Если вы опустите этот аргумент, используемый формат по умолчанию будет соответствовать культуре вашей системы с использованием Culture.Current .

Культура

Необязательный

Это необязательный аргумент, используемый только для устаревших рабочих процессов. Здесь вы можете указать культуру как текстовое значение.

Описание

Date.ToText преобразует dateзначение в его текстовое представление. Дополнительные параметры ( options) в recordформате могут быть предоставлены для определения формата и региональных настроек для преобразования.

Примеры

Давайте рассмотрим несколько практических примеров использования функции Date.ToText:

Базовое преобразование

Вы можете преобразовать #date(2023, 12, 31) в текстовое значение. Полученный результат может варьироваться в зависимости от текущей культуры.

Date.ToText( #date( 2023, 12, 31 ) ) // My machine's output: "31-12-2023"


Копировать

Пользовательские форматы даты

Power Query поддерживает широкий спектр пользовательских форматов дат . Используя необязательный аргумент «options», вы можете указать желаемый формат. Например:

Date.ToText(

#date( 2023, 12, 31 ),

[Format="dd MMM yyyy"]

) // Output: "31 dec 2023"

Зная эти строки форматирования, вы теперь можете создавать свои собственные форматы дат. Например, вы можете вернуть строку, содержащую название дня и описательную дату, следующим образом:

Date.ToText(

#date( 2023, 12, 31 ),

[Format="ddd MMMM d, yyyy"]

) // Output: "Sun December 31, 2023"

Кодекс культуры

Функция Date.ToText также поддерживает код культуры в аргументе options. Вы можете указать код культуры следующим образом:

Date.ToText(

#date( 2023, 12, 31 ),

[Format="MMMM", Culture="nl-NL"]

) // Output: "december"


Копировать

Код культуры может существенно повлиять на результат функции Date.ToText. Например, названия месяцев и форматы дат могут сильно различаться в разных странах. Различия между голландским (nl-NL) и американским (en-US) культурными кодами служат наглядным примером этого.

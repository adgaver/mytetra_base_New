If you want peristency, you can use chromium.launch_persistent_context, see here: playwright.dev/python/docs/api/… This allows you to save the storage state (cookies, etc.) to the local disk and re-use it multiple times.

from playwright.sync_api import sync_playwright

import os

user_dir = '/tmp/playwright'


if not os.path.exists(user_dir):

os.makedirs(user_dir)


with sync_playwright() as p:

browser = p.chromium.launch_persistent_context(user_dir, headless=False)

page = browser.new_page()

page.goto('https://playwright.dev/python', wait_until='domcontentloaded')

VBScript: обработка каждой строки текстового файла


Использование цикла Do ... Loop

Если вам нужно обработать каждую строку текстового файла с помощью VBScript, то вы можете это сделать при простого цикла Do ... Loop. Предположим, есть файл list.txt, который содержит какие-то значения в каждой строке. Создадим в той же директории файл list.vbs, и напишем там следующий скрипт:

Set objFSO = CreateObject("Scripting.FileSystemObject")


Set objFile = objFSO.OpenTextFile("list.txt")

Do Until objFile.AtEndOfStream

Msgbox objFile.ReadLine

Loop

objFile.Close


При выполнении данного скрипта, будет выводиться окно со значением каждой строки, до тех пор, пока содержимое файла не подойдет к концу.

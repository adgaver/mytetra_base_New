Управление клавиатурой в Python

Python - это мощный и гибкий язык программирования, который позволяет разрабатывать различные типы приложений. Один из способов использования Python - это управление клавиатурой. Ниже приведен пример того, как это можно сделать.

Использование библиотеки pynput

pynput - это Python-библиотека, которая позволяет контролировать и отслеживать ввод с клавиатуры и мыши. Установите эту библиотеку, выполнив команду pip install pynput в терминале.


import pynput


from pynput.keyboard import Key, Listener


def on_press(key):

print('{0} pressed'.format(key))


def on_release(key):

print('{0} release'.format(key))

if key == Key.esc:

# Остановить прослушивание

return False


# Собирать события до тех пор, пока не будет нажата клавиша esc

with Listener(on_press=on_press, on_release=on_release) as listener:

listener.join()




В этом примере функция on_press вызывается при нажатии любой клавиши, а функция on_release вызывается при ее отпускании. Программа продолжает слушать клавиатуру, пока не будет нажата клавиша esc.

Отправка нажатий клавиш

Вы также можете использовать Python для симуляции нажатий клавиш. Для этого можно использовать метод Controller из библиотеки pynput.


from pynput.keyboard import Key, Controller


keyboard = Controller()


# Нажмите и отпустите клавишу

keyboard.press(Key.space)

keyboard.release(Key.space)


# Напечатайте строку символов

keyboard.type('Hello World')




В этом примере мы сначала нажимаем и отпускаем пробел, а затем печатаем строку символов 'Hello World'.

Как запустить Playwright в безголовом режиме?

Playwright — это библиотека Node, используемая для автоматизации браузеров Chrome, Safari и Firefox. Эта библиотека предоставляет API высокого уровня для управления этими браузерами. Playwright может работать в безгласном режиме, что означает, что он может выполнять действия в браузере, не отображая пользовательский интерфейс браузера.

Чтобы запустить Playwright в безголовом режиме, вам необходимо установить эту headless опцию true при запуске браузера. По умолчанию Playwright работает в безголовом режиме.

Вот как вы можете это сделать:

В JavaScript:

Копировать

const playwright = require('playwright');

(async () => {

const browser = await playwright.chromium.launch({ headless: true });

const context = await browser.newContext();

const page = await context.newPage();

await page.goto('https://example.com');

await browser.close();

})();



В Питоне:

Копировать

from playwright.sync_api import sync_playwright

with sync_playwright() as p:

browser = p.chromium.launch(headless=True)

context = browser.new_context()

page = context.new_page()

page.goto("https://example.com")

browser.close()



В обоих примерах headlessдля этого параметра установлено значение, trueчто означает, что браузер будет работать в безгласном режиме. Если вы хотите отобразить пользовательский интерфейс браузера, вы можете установить headlessдля этого параметра значение false.

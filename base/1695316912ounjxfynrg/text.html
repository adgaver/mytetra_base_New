Emulate viewport size

Playwright открывает окно браузера с установленной областью просмотра определенной ширины и высоты и не реагирует, поскольку тесты необходимо запускать в тех же условиях. Используйте --viewportопцию для создания тестов с другим размером области просмотра.

 

npx playwright codegen --viewport-size=800,600 playwright.dev

 

browser = playwright.chromium.launch(headless=False, executable_path=str_path_browser)

context = browser.new_context(viewport={"width": 1280, "height": 720})

Хотя BeautifulSoup по умолчанию поддерживает парсер HTML. Если вы хотите использовать какие-либо другие сторонние парсеры Python, вам необходимо установить этот внешний парсер, например (lxml).

soup_object= BeautifulSoup(markup, "html.parser") #Python HTML parser

Но если вы не указали ни один парсер в качестве параметра, вы получите предупреждение о том, что парсер не указан.

soup_object= BeautifulSoup(markup) #Warnning

Чтобы использовать любой другой внешний парсер, вам необходимо его установить, а затем указать его. нравиться

pip install lxmlsoup_object= BeautifulSoup(markup, 'lxml') # C dependent parser

Внешний парсер зависит от C и Python, что может иметь некоторые преимущества и недостатки.

Как посмотреть функции в Python?

В Python есть несколько способов, с помощью которых вы можете посмотреть функции в вашем коде:

  1. Использование функции dir() для объекта: Вы можете использовать встроенную функцию dir() для объекта и посмотреть все атрибуты этого объекта, включая методы (функции), определенные для него. Например:

    import math

    print(dir(math)) # Покажет все функции и атрибуты, связанные с модулем math


  2. Использование модуля inspect: Модуль inspect позволяет получить информацию о объектах Python, включая функции. Вы можете использовать его для получения списка функций, определенных в модуле или классе. Например:

    import inspect


    def my_function():

    pass


    functions_list = [func for func in dir() if inspect.isfunction(eval(func))]

    print(functions_list) # Покажет все функции в текущей области видимости


  3. Использование инструментов сторонних библиотек: Существуют сторонние библиотеки, такие как pydoc, которые предоставляют более расширенные возможности для просмотра и документирования функций в вашем коде.

Эти методы помогут вам просмотреть функции, определенные в вашем коде, и исследовать их свойства и атрибуты.

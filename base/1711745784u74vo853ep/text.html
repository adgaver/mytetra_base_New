Перейти на один уровень папки

Поскольку путь может не соответствовать текущему рабочему каталогу, вам необходимо извлечь путь из строки.

Найдите последнюю \ и прочитайте все символы слева:

ParentPath = Left$(Path, InStrRev(Path, "\"))

Если вы работаете с текущим каталогом, это ChDir ".." поднимет вас на один уровень выше, новый путь может быть возвращен с помощью CurrDir.


Самый надежный способ сделать это - использовать скриптинг.FileSystemObject . У него есть метод, который получит родительскую папку без попыток ее анализа.:

Function fn_path_parrent(ByVal str_path As String) As String
'1 --------------------------------------------------
    If InStr(1, str_path, "/", vbTextCompare) Then
        str_separator = "/"
    Else
        str_separator = "\"
    End If
    fn_path_parrent = Left$(str_path, InStrRev(str_path, str_separator) - 1)


'2 --------------------------------------------------
    With CreateObject("Scripting.FileSystemObject")
        fn_path_parrent = .GetParentFolderName(str_path)
    End With
End Function

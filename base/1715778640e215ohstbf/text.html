Добавление нового столбца к существующему фрейму данных в Pandas

Последнее обновление : 21 декабря 2023 г.

Улучшить

Добавление новых столбцов к существующему фрейму данных является фундаментальной задачей при анализе данных с использованием Pandas. Это позволяет обогатить ваши данные дополнительной информацией и облегчить дальнейший анализ и манипулирование. В этой статье будут рассмотрены различные методы добавления новых столбцов, включая простое назначение, insert() метод, assign() метод. Давайте обсудим добавление новых столбцов в существующий фрейм данных Pandas.

Что такое фрейм данных Pandas?

Фрейм данных Pandas представляет собой двумерную, изменяемую по размеру, потенциально разнородную табличную структуру данных с помеченными осями (строками и столбцами). Это фундаментальная структура данных в экосистеме Python data science, которая обеспечивает мощный способ работы с табличными данными.

Вот несколько ключевых функций фрейма данных Pandas:

Представление данных: Хранит данные в табличном формате со строками и столбцами.

Разнородные типы данных: В разных столбцах могут содержаться разные типы данных (например, целые числа, числа с плавающей запятой, строки, логические значения).

Маркировка: У каждой строки и столбца есть метка (индекс и имена столбцов).

Изменяемый: Позволяет манипулировать данными и изменять их.

Мощные операции: Предоставляет различные функции и методы для анализа, манипулирования и исследования данных.

Расширяемый: Можно настраивать и расширять дополнительными функциями с помощью библиотек и пользовательских функций.

Добавление нового столбца к существующему фрейму данных в Pandas на Python

Существует несколько способов добавить новый столбец к существующему фрейму данных в Pandas на Python:


  • Создание примера фрейма данных
  • С помощью Dataframe.insert() метода
  • С помощью Dataframe.assign() метода
  • Использование словаря
  • Использование списка
  • Использование .loc()
  • Добавление более одного столбца в существующий фрейм данных

Создание примера фрейма данных

Здесь мы создаем образец фрейма данных:

Python3


import pandas as pd

 

data = {'Name': ['Jai', 'Princi', 'Gaurav', 'Anuj'],

        'Height': [5.1, 6.2, 5.1, 5.2],

        'Qualification': ['Msc', 'MA', 'Msc', 'Msc']}

 

df = pd.DataFrame(data)

print(df)

Вывод:

Name Height Qualification

0 Jai 5.1 Msc

1 Princi 6.2 MA

2 Gaurav 5.1 Msc

3 Anuj 5.2 Msc



Обратите внимание, что длина вашего списка должна соответствовать длине столбца индекса, в противном случае будет выдана ошибка.

Добавьте новый столбец в существующий фрейм данных с помощью DataFrame.insert()

Это дает свободу добавлять столбец в любом месте, которое нам нравится, а не только в конце. Это также предоставляет различные варианты вставки значений столбца.

Python3


import pandas as pd

 

# Define a dictionary containing Students data

data = {'Name': ['Jai', 'Princi', 'Gaurav', 'Anuj'],

        'Height': [5.1, 6.2, 5.1, 5.2],

        'Qualification': ['Msc', 'MA', 'Msc', 'Msc']}

 

# Convert the dictionary into DataFrame

df = pd.DataFrame(data)

 

# Using DataFrame.insert() to add a column

df.insert(2, "Age", [21, 23, 24, 21], True)

 

# Observe the result

print(df)

Вывод: 

Name Height Age Qualification

0 Jai 5.1 21 Msc

1 Princi 6.2 23 MA

2 Gaurav 5.1 24 Msc

3 Anuj 5.2 21 Msc




Добавление столбцов в фрейм данных Pandas с помощью Dataframe.assign()

Этот метод создаст новый фрейм данных с добавлением нового столбца к старому фрейму данных.

Python3


import pandas as pd

 

# Define a dictionary containing Students data

data = {'Name': ['Jai', 'Princi', 'Gaurav', 'Anuj'],

        'Height': [5.1, 6.2, 5.1, 5.2],

        'Qualification': ['Msc', 'MA', 'Msc', 'Msc']}

 

 

# Convert the dictionary into DataFrame

df = pd.DataFrame(data)

 

# Using 'Address' as the column name and equating it to the list

df2 = df.assign(address=['Delhi', 'Bangalore', 'Chennai', 'Patna'])

 

print(df2)

Вывод: 

Name Height Qualification address

0 Jai 5.1 Msc Delhi

1 Princi 6.2 MA Bangalore

2 Gaurav 5.1 Msc Chennai

3 Anuj 5.2 Msc Patna




Pandas Добавляет столбец к фрейму данных с помощью словаря

Мы можем использовать словарь Python для добавления нового столбца в фрейм данных pandas. Используйте существующий столбец в качестве ключевых значений, и их соответствующие значения будут значениями для нового столбца.

Python3


# Import pandas package

import pandas as pd

 

# Define a dictionary containing Students data

data = {'Name': ['Jai', 'Princi', 'Gaurav', 'Anuj'],

        'Height': [5.1, 6.2, 5.1, 5.2],

        'Qualification': ['Msc', 'MA', 'Msc', 'Msc']}

 

# Define a dictionary with key values of

# an existing column and their respective

# value pairs as the # values for our new column.

address = {'Delhi': 'Jai', 'Bangalore': 'Princi',

           'Patna': 'Gaurav', 'Chennai': 'Anuj'}

 

# Convert the dictionary into DataFrame

df = pd.DataFrame(data)

 

# Provide 'Address' as the column name

df['Address'] = address

 

# Observe the output

print(df)

Вывод: 

Name Height Qualification Address

0 Jai 5.1 Msc Delhi

1 Princi 6.2 MA Bangalore

2 Gaurav 5.1 Msc Chennai

3 Anuj 5.2 Msc Patna

В этом примере, Pandas добавляет новые столбцы из списка “Адрес” в существующий фрейм данных Pandas, используя словарь и список.

Python3


# Declare a list that is to be converted into a column

address = ['Delhi', 'Bangalore', 'Chennai', 'Patna']

 

# Using 'Address' as the column name

df['Address'] = address

 

print(df)

Вывод:

Добавьте новый столбец в существующий фрейм данных Pandas с помощью Dataframe.loc()

В этом примере создается фрейм данных Pandas с именем df со столбцами “Name”, “Height” и “Qualification” и добавляется новый столбец “Address” с использованием loc атрибута.

Python3


import pandas as pd

 

data = {'Name': ['Jai', 'Princi', 'Gaurav', 'Anuj'],

       'Height': [5.1, 6.2, 5.1, 5.2],

       'Qualification': ['Msc', 'MA', 'Msc', 'Msc']}

 

df = pd.DataFrame(data)

 

# Create the list of new column values

address = ["Delhi", "Bangalore", "Chennai", "Patna"]

 

# Add the new column using loc

df.loc[:, "Address"] = address

 

print(df)

Вывод:

Name Height Qualification Address

0 Jai 5.1 Msc Delhi

1 Princi 6.2 MA Bangalore

2 Gaurav 5.1 Msc Chennai

3 Anuj 5.2 Msc Patna




Добавление более одного столбца в существующий фрейм данных

В этом примере существующий фрейм данных Pandas расширяется df двумя новыми столбцами: “Возраст” и “Состояние”, используя соответствующие списки данных.

Python3


import pandas as pd

 

data = {'Name': ['Jai', 'Princi', 'Gaurav', 'Anuj'],

        'Height': [5.1, 6.2, 5.1, 5.2],

        'Qualification': ['Msc', 'MA', 'Msc', 'Msc'],

        'Address': ['Delhi', 'Bangalore', 'Chennai', 'Patna']}

 

df = pd.DataFrame(data)

 

# Define new data for additional columns

age = [22, 25, 23, 24]

state = ['NCT', 'Karnataka', 'Tamil Nadu', 'Bihar']

 

# Add multiple columns using dictionary assignment

new_data = {'Age': age, 'State': state }

df = df.assign(**new_data)

 

print(df)

Вывод:

Name Height Qualification Address Age State

0 Jai 5.1 Msc Delhi 22 NCT

1 Princi 6.2 MA Bangalore 25 Karnataka

2 Gaurav 5.1 Msc Chennai 23 Tamil Nadu

3 Anuj 5.2 Msc Patna 24 Bihar

Работа с Pandas и XlsxWriter 

Python Pandas — это библиотека анализа данных Python. Она может читать, фильтровать и переупорядочивать малые и большие наборы данных и выводить их в различных форматах, включая Excel.

Pandas записывает файлы Excel xlsx с помощью openpyxl или XlsxWriter.

Использование XlsxWriter с Pandas 

Чтобы использовать XlsxWriter с Pandas, укажите его в качестве движка записи Excel :

import pandas as pd


# Create a Pandas dataframe from the data.

df = pd.DataFrame({'Data': [10, 20, 30, 20, 15, 30, 45]})


# Create a Pandas Excel writer using XlsxWriter as the engine.

writer = pd.ExcelWriter('pandas_simple.xlsx', engine='xlsxwriter')


# Convert the dataframe to an XlsxWriter Excel object.

df.to_excel(writer, sheet_name='Sheet1')


# Close the Pandas Excel writer and output the Excel file.

writer.close()

Доступ к XlsxWriter из Pandas 

Чтобы применить функции XlsxWriter, такие как Charts, Conditional Formatting и Column Formatting к выходным данным Pandas, нам нужно получить доступ к базовым объектам workbook и worksheet . После этого мы можем обращаться с ними как с обычными объектами XlsxWriter.

Продолжая приведенный выше пример, сделаем это следующим образом:

import pandas as pd


# Create a Pandas dataframe from the data.

df = pd.DataFrame({'Data': [10, 20, 30, 20, 15, 30, 45]})


# Create a Pandas Excel writer using XlsxWriter as the engine.

writer = pd.ExcelWriter('pandas_simple.xlsx', engine='xlsxwriter')


# Convert the dataframe to an XlsxWriter Excel object.

df.to_excel(writer, sheet_name='Sheet1')


# Get the xlsxwriter objects from the dataframe writer object.

workbook = writer.book

worksheet = writer.sheets['Sheet1']



Это эквивалентно следующему коду при использовании XlsxWriter отдельно:

workbook = xlsxwriter.Workbook('filename.xlsx')

worksheet = workbook.add_worksheet()



Затем объекты Workbook и Worksheet можно использовать для доступа к другим функциям XlsxWriter, см. ниже.

Добавление диаграмм в вывод Dataframe 

Получив объекты Workbook и Worksheet, как показано в предыдущем разделе, мы можем использовать их для применения других функций, таких как добавление диаграммы:

# Get the xlsxwriter objects from the dataframe writer object.

workbook = writer.book

worksheet = writer.sheets['Sheet1']


# Create a chart object.

chart = workbook.add_chart({'type': 'column'})


# Get the dimensions of the dataframe.

(max_row, max_col) = df.shape


# Configure the series of the chart from the dataframe data.

chart.add_series({'values': ['Sheet1', 1, 1, max_row, 1]})


# Insert the chart into the worksheet.

worksheet.insert_chart(1, 3, chart)

Форматирование вывода Dataframe 

XlsxWriter и Pandas предоставляют очень ограниченную поддержку для форматирования выходных данных из dataframe, за исключением форматирования по умолчанию, такого как ячейки заголовка и индекса, а также любые ячейки, содержащие даты или даты и времени. Кроме того, невозможно форматировать любые ячейки, к которым уже применен формат по умолчанию.

Если вам требуется очень контролируемое форматирование выходных данных dataframe, то вам, вероятно, лучше использовать Xlsxwriter напрямую с необработанными данными, взятыми из Pandas. Однако некоторые параметры форматирования доступны.

Например, можно установить форматы даты и времени по умолчанию через интерфейс Pandas:

writer = pd.ExcelWriter("pandas_datetime.xlsx",

engine='xlsxwriter',

datetime_format='mmm d yyyy hh:mm:ss',

date_format='mmmm dd yyyy')

Можно отформатировать любые другие данные столбца, отличные от даты/даты-времени, используя set_column():

# Add some cell formats.

format1 = workbook.add_format({'num_format': '#,##0.00'})

format2 = workbook.add_format({'num_format': '0%'})


# Set the column width and format.

worksheet.set_column(1, 1, 18, format1)


# Set the format but not the column width.

worksheet.set_column(2, 2, None, format2)

Форматирование заголовков Dataframe 

Pandas записывает заголовок dataframe с форматом ячейки по умолчанию. Поскольку это формат ячейки, его нельзя переопределить с помощью set_row(). Если вы хотите использовать свой собственный формат для заголовков, то лучшим подходом будет отключить автоматический заголовок из Pandas и написать свой собственный. Например:

# Turn off the default header and skip one row to allow us to insert a

# user defined header.

df.to_excel(writer, sheet_name='Sheet1', startrow=1, header=False)


# Get the xlsxwriter workbook and worksheet objects.

workbook = writer.book

worksheet = writer.sheets['Sheet1']


# Add a header format.

header_format = workbook.add_format({

'bold': True,

'text_wrap': True,

'valign': 'top',

'fg_color': '#D7E4BC',

'border': 1})


# Write the column headers with the defined format.

for col_num, value in enumerate(df.columns.values):

worksheet.write(0, col_num + 1, value, header_format)



Полный пример см. в разделе Пример: Вывод Pandas Excel с пользовательским форматом заголовка .

Добавление Dataframe в таблицу рабочего листа 

Как объясняется в статье Работа с таблицами рабочих листов , таблицы в Excel — это способ группировки диапазона ячеек в единый объект, например:

Чтобы сделать это с помощью фрейма данных Pandas, сначала нужно записать данные без индекса или заголовка, а затем начать на одну строку вперед, чтобы освободить место для заголовка таблицы:

df.to_excel(writer, sheet_name='Sheet1',

startrow=1, header=False, index=False)



Затем мы создаем список заголовков для использования в add_table():

column_settings = [{'header': column} for column in df.columns]



Наконец, мы добавляем структуру таблицы Excel на основе формы фрейма данных и с заголовками столбцов, которые мы сгенерировали из столбцов фрейма данных:

(max_row, max_col) = df.shape


worksheet.add_table(0, 0, max_row, max_col - 1, {'columns': column_settings})

Добавление автофильтра к выходным данным Dataframe 

Как объясняется в статье Работа с автофильтрами , автофильтры в Excel — это способ фильтрации двумерного диапазона данных для отображения только тех строк, которые соответствуют заданным пользователем критериям.

Чтобы сделать это с помощью фреймворка данных Pandas, сначала нужно записать данные без индекса (если только вы не хотите включить его в отфильтрованные данные):

df.to_excel(writer, sheet_name='Sheet1', index=False)



Затем мы получаем форму фрейма данных и добавляем автофильтр:

worksheet.autofilter(0, 0, max_row, max_col - 1)



Мы также можем добавить необязательный критерий фильтра. Заполнитель «Region» в фильтре игнорируется и может быть любой строкой, которая добавляет ясности выражению:

worksheet.filter_column(0, 'Region == East')



Однако недостаточно просто применить критерии. Строки, которые не соответствуют, также должны быть скрыты. Мы используем Pandas, чтобы определить, какие строки следует скрыть:

for row_num in (df.index[(df['Region'] != 'East')].tolist()):

worksheet.set_row(row_num + 1, options={'hidden': True})

Обработка нескольких Pandas Dataframes 

Можно записать более одного dataframe на рабочий лист или на несколько рабочих листов. Например, чтобы записать несколько dataframe на несколько рабочих листов:

# Write each dataframe to a different worksheet.

df1.to_excel(writer, sheet_name='Sheet1')

df2.to_excel(writer, sheet_name='Sheet2')

df3.to_excel(writer, sheet_name='Sheet3')



Полный пример см. в разделе Пример: Pandas Excel с несколькими фреймами данных .

Также можно разместить несколько фреймов данных на одном рабочем листе:

# Position the dataframes in the worksheet.

df1.to_excel(writer, sheet_name='Sheet1') # Default position, cell A1.

df2.to_excel(writer, sheet_name='Sheet1', startcol=3)

df3.to_excel(writer, sheet_name='Sheet1', startrow=6)


# Write the dataframe without the header and index.

df4.to_excel(writer, sheet_name='Sheet1',

startrow=7, startcol=4, header=False, index=False)



Полный пример см. в разделе Пример: позиционирование фрейма данных Pandas Excel .

Передача параметров конструктора XlsxWriter в Pandas 

XlsxWriter поддерживает несколько Workbook()опций конструктора, таких как strings_to_urls(). Их также можно применить к Workbookобъекту, созданному Pandas, используя engine_kwargsключевое слово:

writer = pd.ExcelWriter('pandas_example.xlsx',

engine='xlsxwriter',

engine_kwargs={'options': {'strings_to_numbers': True}})



Обратите внимание, что версии Pandas до 1.3.0 использовали следующий синтаксис:

writer = pd.ExcelWriter('pandas_example.xlsx',

engine='xlsxwriter',

options={'strings_to_numbers': True})



Сохранение выходных данных Dataframe в строку 

Также возможно записать вывод Pandas XlsxWriter DataFrame в массив байтов:

import pandas as pd

import io


# Create a Pandas dataframe from the data.

df = pd.DataFrame({'Data': [10, 20, 30, 20, 15, 30, 45]})


output = io.BytesIO()


# Use the BytesIO object as the filehandle.

writer = pd.ExcelWriter(output, engine='xlsxwriter')


# Write the data frame to the BytesIO object.

df.to_excel(writer, sheet_name='Sheet1')


writer.close()

xlsx_data = output.getvalue()


# Do something with the data...



Примечание: для этой функции требуется Pandas >= 0.17.

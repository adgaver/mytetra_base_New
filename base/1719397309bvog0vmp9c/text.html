Какие типы селекторов доступны в Playwright?

В Playwright селекторы используются для поиска элементов на веб-странице. Playwright поддерживает широкий спектр селекторов, упрощающих выбор элементов. Вот различные типы селекторов, доступные в Playwright:

  • Селекторы CSS . Селекторы CSS широко используются в веб-разработке для стилизации элементов. Их также можно использовать в Playwright для выбора элементов.

Копировать

const elementHandle = await page.$('css=div > span');



  • Селекторы XPath : XPath — это язык, используемый для выбора узлов в XML-документе. Его также можно использовать для поиска элементов в HTML.

Копировать

const elementHandle = await page.$('xpath=//div/span');



  • Селекторы текста : Селекторы текста позволяют выбирать элементы по их текстовому содержимому.

Копировать

const elementHandle = await page.$('text="My Button"');



  • Селекторы идентификаторов : селекторы идентификаторов позволяют выбирать элемент по его идентификатору.

Копировать

const elementHandle = await page.$('id=my-button');



  • Селекторы атрибутов данных : эти селекторы позволяют выбирать элементы по их атрибутам данных.

Копировать

const elementHandle = await page.$('data-testid=my-button');



  • Селекторы JSHandle : эти селекторы позволяют выбирать элементы с помощью JavaScript.

Копировать

const jsHandle = await page.evaluateHandle(() => document.querySelector('button'));



  • Селекторы тегов . Селекторы тегов позволяют выбирать элемент по его имени тега.

Копировать

const elementHandle = await page.$('button');



  • Селекторы атрибутов . Селекторы атрибутов позволяют выбирать элемент по его атрибуту.

Копировать

const elementHandle = await page.$('[my-attribute]');



  • Селекторы классов : Селекторы классов позволяют выбрать элемент по его классу.

Копировать

const elementHandle = await page.$('.my-class');



  • Селекторы Playwright : Playwright также поддерживает собственный механизм выбора, который сочетает в себе CSS, XPath и селекторы текста.

Копировать

const elementHandle = await page.$('css=div >> xpath=//span >> text="My Button"');



Каждый из этих селекторов имеет свои варианты использования и преимущества. В зависимости от сложности веб-страницы и конкретных элементов, на которые вам нужно ориентироваться, вы можете выбрать наиболее подходящий селектор.

Python объединил несколько файлов Excel XLS, формат XLSX

Теги:  python база данных сценарий

использоватьpandasМодуль иglob, Я делал это с XLRD раньше, код длинный и может читать только формат XLS. После обучения Pandas оптимизировано, код короткий и может быть объединен с XLSX вместе с XLSX вместе вместе

Простой код версии заключается в следующем

import pandas as pd

import glob


filelocation = 'D:\\file\\' #Folder позиция

li=[]


# Читать все файлы под папкой (XLS и XLSX читают)

for i in glob.glob(filelocation+"*.*"):

li.append(pd.read_excel(i))


# Расположение файла Excel

filedestination="D:\\Documents and Settings\\"


numbers = len(li)

names = 'test'

writer = pd.ExcelWriter(filedestination+names+".xlsx")

pd.concat(li).to_excel(writer,'Sheet1',index=False)


print('\ n%D Файл был успешно объединен! '% numbers)

writer.save()

VBScript учебник

VBScript является языком сценариев Microsoft.

VBScript является облегченной версией языка программирования от Microsoft Visual Basic.

VBScript является языком сценариев по умолчанию в ASP (Active Server Pages) .




VBScript на сервере

Когда VBScript используется с ASP, утверждение response. write() response. write() производит вывод.

Нажмите на "Show Example" кнопку , чтобы увидеть , как это работает:

пример

<html>
<body>

<%response.write("This is my first VBScript")%>

</body>
</html>

Что такое VBScript?

  • VBScript является языком сценариев
  • Язык сценариев представляет собой легкий язык программирования
  • VBScript является облегченной версией языка программирования от Microsoft Visual Basic
  • VBScript является языком по умолчанию в ASP (Active Server Pages)

VBScript в браузере?

VBScript следует использовать в качестве языка сценариев на стороне сервера. Он должен not быть использован для написания сценариев в browser .

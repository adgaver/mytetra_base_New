15 суперполезных горячих клавиш в Windows 11. Вы будете пользоваться ими каждый день и жалеть, что не знали о них раньше

Александр Кузнецов — 27 сентября 2023, 19:50
11344



Работа с компьютером станет эффективнее, если вы будете использовать горячие сочетания клавиш. В Windows 11 их несколько десятков, а мы по мотивам публикации The Verge подобрали самые полезные.

Увеличение экрана

Хотите разглядеть что-нибудь поближе? Необязательно доставать лупу, как человек из известных мемов, можно воспользоваться специальной утилитой. Нажмите Win и +, это приблизит изображение на экране, а Win и - вернёт к исходному значению. Для навигации по увеличенному изображению можно пользоваться мышью или тачпадом.

Просмотр приложений на разных рабочих столах

Всем известно сочетание Alt + Tab, которое позволяет быстро переключаться между запущенными программами. Продвинутая версия этой возможности запускается кнопками Win + Tab: если у вас несколько виртуальных рабочих столов (например, один для работы, другой для развлечений), можно увидеть их все и переключаться между открытыми в них приложениями.

История буфера обмена

По умолчанию буфер обмена содержит лишь данные, скопированные в него последними. По нажатию Win + V откроется утилита, которая покажет историю буфера обмена; любую информацию можно скопировать оттуда, чтобы снова вставить куда-нибудь.

Быстрый переход к рабочему столу

Win + D сворачивает все окна и показывает только рабочий стол с размещёнными на нём иконками (если они там есть и не скрыты). Повторное нажатие тех же кнопок возвращает все активные окна обратно.

Обновление страницы в браузере

Для обновления содержимого на странице нет необходимости нажимать на кнопку рефреша в самом браузере, это можно сделать кнопкой F5 или сочетанием Ctrl + R. На некоторых клавиатурах (например, в ноутбуках), функциональные клавиши зарезервированы под другие действия, поэтому может потребоваться нажать сразу две кнопки — Fn + F5.

Вернуть закрытую страницу

Почти все браузеры поддерживают повторное открытие закрытых недавно страниц — для этого можно использовать Ctrl + Shift + T.

Увеличение содержимого в «Проводнике»

Ctrl + Alt и кнопка «+» увеличивают иконки в «Проводнике», Ctrl + Alt и кнопка «-» уменьшают их. Такое же действие поддерживается некоторыми другими приложениями. Как вариант, можно использовать Ctrl и колёсико мыши вверх или вниз — это работает и с сайтами в браузерах, причём программы запоминают настройки зума.

Скроллинг

Браузеры и некоторые другие программы позволяют листать страницу вниз нажатием пробела, это широко известный факт. Если зажать Shift перед нажатием на пробел, страница листнётся не вниз, а вверх.

Полноэкранные приложения

Некоторые программы (например, браузеры) поддерживают работу в полноэкранном режиме без лишних меню и прочих элементов. Обычно этот режим включается нажатием F11 (или Fn + F11 на некоторых клавиатурах), выйти из него можно той же клавишей.

Меню «Пуск»

Чтобы открыть меню «Пуск», не обязательно нажимать на него курсором мыши, можно воспользоваться сочетанием Ctrl + Esc. А для просмотра «скрытого» меню, которое отображается по нажатию на «Пуск» правой кнопкой мыши, можно использовать Win + X.

Скриншоты

Можно пропустить появление утилиты «Ножницы», которая запускается каждый раз, когда вы делаете скриншоты нажатием PrtSc, для этого нужно нажимать Win + PrtSc. В поддиректорию «Скриншоты» в папке «Изображения» сохраняется PNG-файл с копией содержимого целого экрана, картинка также копируется в буфер обмена. Alt + PrtSc захватывает активное окно и копирует его изображение в буфер, откуда его можно вставить в поддерживаемые приложения кнопками Ctrl + V.

Навигация в «Проводнике»

Проводник поддерживает несколько сочетаний клавиш для быстрой навигации. Например, Backspace или Alt + кнопка со стрелкой налево возвращают вас к предыдущей открытой директории, Alt + кнопка со стрелкой направо возвращают к той папке, с которой вы перешли обратно, а Alt + кнопка вверх позволяет перейти в родительскую папку.

Пропуск «Корзины»

Shift + Del удаляет файлы без возможности восстановления из «Корзины». Пользуйтесь с осторожностью!

Переименование файлов

Для переименования файла в «Проводнике» выделите этот файл, нажмите F2 (или Fn + F2 на некоторых клавиатурах) и задайте ему новое имя. Если выделить несколько файлов и нажать F2, получится переименовать сразу их все, к общему имени будут добавлены цифры в скобках.

Отменить отмену

Клавиши Ctrl + Z позволяют в некоторых приложениях отменить последнее действие (а повторные нажатия — предыдущие действия, иногда даже после сохранения файла). С помощью Ctrl + Y можно вернуть отменённые изменения.

Python: Сделать окно активным

В этой статье мы рассмотрим, как использовать Python для активации окон на вашем компьютере.

PyGetWindow и pyautogui

Для начала нам потребуются две библиотеки: PyGetWindow и pyautogui. Установите их с помощью следующей команды:

pip install pygetwindow pyautogui


Активация окна по его заголовку

Теперь давайте напишем простой код, который активирует окно по его заголовку:

import pygetwindow as gw

import pyautogui


title = 'Блокнот'

try:

window = gw.getWindowsWithTitle(title)[0]

window.activate()

except IndexError:

print('Окно с указанным заголовком не найдено.')


В этом примере мы ищем окно с заголовком "Блокнот" и активируем его. Если такого окна не существует, выводится сообщение об ошибке.

Активация окна по его идентификатору

Иногда удобнее активировать окно по его уникальному идентификатору (HWND). Вот как это сделать:

import pygetwindow as gw

import pyautogui


hwnd = 123456

try:

window = gw.getWindowsWithHWND(hwnd)[0]

window.activate()

except IndexError:

print('Окно с указанным идентификатором не найдено.')


Замените "hwnd" на идентификатор окна, которое вы хотите активировать.

Ограничения и предостережения

Обратите внимание, что эти методы могут не работать для всех окон. В некоторых случаях окна могут быть защищены от автоматической активации.

Будьте внимательны при использовании этого кода, так как активация окна может привести к нежелательным последствиям, если пользователь в это время выполняет другие действия на компьютере.

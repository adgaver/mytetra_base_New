Как отформатировать дату с помощью strftime() в Python?

Последнее обновление : 05 декабря 2022 г.

Улучшить

В этой статье мы увидим, как отформатировать дату с помощью strftime() в Python. localtime() и gmtime() возвращает кортеж, представляющий время, и этот кортеж преобразуется в строку, указанную аргументом format, с использованием метода времени python strftime().

Синтаксис:

time.strftime(формат[, сек])

секунда: Это время в количестве секунд, которое необходимо отформатировать.

формат: Это директива или код формата, который будет использоваться для форматирования.

Директива или код форматирования

Возвращенное значение

Пример

%Y

Полный год с указанием century

2021,2022

%y

Год без столетия с нулевым дополненным значением

00,01,….21,22…,99

%-y

Год без столетия

0,1…,99

%m

Месяц с дополненным нулевым значением

01-12

%-m

Месяц без добавленного нулевого значения

1-12

%B

Полное название месяца

Январь, февраль, ..., декабрь

%b

Краткая форма месяца

Январь, февраль, ..., декабрь

%A

Полное название дня недели

Воскресенье, понедельник,..

%a

Краткая форма названия дня недели

Вс, Пн,..

%w

День недели как десятичное значение

0-6

%d

Дни с дополненным нулевым значением

01-31

%-d

Дни с десятичным значением

1-31

%H

Час (24-часовые часы) как значение, дополненное нулем.

00-23

%-H

Час (24-часовые часы) без добавленного к нулю значения.

0,1,…,23

%I

Час (12-часовые часы) как значение, дополненное нулем.

01-12

%-Я

Час (12-часовые часы) без добавленного к нулю значения.

1-12

%M

Минуты с добавлением нуля

00-59

%-M

Минуты без добавленного нуля

0-59

%S

Секунды с нулевым дополненным значением

00-59

%-S

Секунды без добавленного нулевого значения

0-59

%f

Микросекунды с добавлением нуля

000000 – 999999

%p

Язык - AM или PM.

УТРОМ / ВЕЧЕРОМ

%j

День года с дополненным нулевым значением

001-366

%-j

День года без добавленного нулевого значения

1-366

%z

Смещение UTC в виде +HHMM или -HHMM.

 

%Z

Название часового пояса.

 

%C

Дата и время, соответствующие локали

Пт, 02 апр, 02:09:07 2020

%x

Дата, соответствующая локали

02/04/22

%X

Время, соответствующее локали

02:04:22

%W

Номер недели в году. Понедельник как первый день недели

00-53

%U

Номер недели в году. Воскресенье как первый день недели

00-53

Ниже приведены несколько примеров для лучшего понимания.

Пример 1:

  • Python3


from datetime import datetime

 

# current time and date

# datetime object

time = datetime.now()

print("Without formatting:", time)

 

# formatting date using strftime

print("After formatting:", time.strftime("%b %d, %Y"))

Вывод

Without formatting: 2022-11-29 12:59:08.088819

After formatting: Nov 29, 2022

# formatting date using strftime

print("Month name", time.strftime("%B"))

Пример 3:

  • Python3


from datetime import datetime

 

# current time and date

# datetime object

time = datetime.now()

 

# formatting date using strftime

# format = MM/DD/YY

print(time.strftime("%m/%d/%y"))

 

# format = Month D, Yr

print(time.strftime("%B %d, %Y"))

 

# time formatting

# HH:MM:SS

print(time.strftime("%H:%M:%S"))

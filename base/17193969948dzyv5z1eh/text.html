Как обрабатывать действия клавиатуры в Playwright?

Playwright предлагает несколько методов имитации ввода с клавиатуры, таких как ввод текста, нажатие клавиш или удерживание клавиш. В этой статье мы обсудим два ключевых метода: page.type()и page.press().

страница.тип()

Этот метод позволяет имитировать ввод текста в поле ввода. Он принимает два аргумента: селектор поля ввода и текст, который вы хотите ввести.

Вот пример Python:

Копировать

from playwright.sync_api import sync_playwright

with sync_playwright() as p:

browser = p.chromium.launch()

page = browser.new_page()

page.goto('https://example.com')

page.type('#input_field_id', 'Hello, World!')

browser.close()



Вот пример JavaScript:

Копировать

const playwright = require('playwright');

(async () => {

const browser = await playwright.chromium.launch();

const context = await browser.newContext();

const page = await context.newPage();

await page.goto('https://example.com');

await page.type('#input_field_id', 'Hello, World!');

await browser.close();

})();



страница.press()

Этот метод позволяет имитировать нажатие клавиши на клавиатуре. Он принимает два аргумента: селектор поля ввода и клавишу, которую вы хотите нажать.

Вот пример Python:

Копировать

from playwright.sync_api import sync_playwright

with sync_playwright() as p:

browser = p.chromium.launch()

page = browser.new_page()

page.goto('https://example.com')

page.click('#input_field_id') # focus the input field

page.press('#input_field_id', 'Enter') # Press 'Enter'

browser.close()



Вот пример JavaScript:

Копировать

const playwright = require('playwright');

(async () => {

const browser = await playwright.chromium.launch();

const context = await browser.newContext();

const page = await context.newPage();

await page.goto('https://example.com');

await page.click('#input_field_id'); // focus the input field

await page.press('#input_field_id', 'Enter'); // Press 'Enter'

await browser.close();

})();



В этих примерах замените #input_field_idфактический селектор вашего поля ввода и 'Hello, World!'фактический текст, который вы хотите ввести.

Вы также можете имитировать другие действия с клавиатурой, такие как удержание клавиши, отпускание клавиши и т. д., используя другие методы, предоставляемые Playwright. Более подробную информацию можно найти в официальной документации Playwright .

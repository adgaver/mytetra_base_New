Заготовка для сетевого приложения Access

Автор Alex   

06.05.2014 г.

Пример представляет собой заготовку для сетевого приложения Access.

Возможности:

  • подключение к выбранному файлу базы данных (файл с таблицами). Путь к базе данных запоминается, т.е. если с момента последнего запуска имя и расположение файла базы данных не менялось - таблицы подключатся автоматически, о чем будет выведено соответствующее сообщение. Если имя или расположение файла базы данных изменилось - будет выведено сообщение об ошибке подключения и откроется форма с настройками для уточнения пути к файлу базы данных.
  • после запуска файла интерфейса, в заголовке окна будет отображено имя файла интерфейса и полный путь к файлу базы данных.
  • при запуске файла интерфейса, автоматически создается резервная копия файла интерфейса и файла подключенной базы данных, вида <ИмяФайла_Дата(ГГГГММДД)_Время(ЧЧММ)>. Резервные копии создаются в каталогах расположения, соответственно, файла базы данных и файла интерфейса.


Примечания:

  • имена файлов интерфейса, базы данных, а так же каталоги их расположения могут быть любыми.
  • при запуске файла интерфейса в скрытом режиме открывается форма "УстановкиЛокальные" и остается открытым в течение всего сеанса работы.

Часто при работе с временем в Python возникает необходимость вывести временной интервал в формате, который не содержит доли секунд. В стандартной библиотеке Python для работы с временем используется модуль datetime. В этой статье мы рассмотрим способы вывода интервала времени без долей секунд с помощью модуля datetime.

Первый способ вывода интервала времени без долей секунд — это использование метода strftime() объекта timedelta. Метод strftime() принимает строку формата времени и возвращает строку, содержащую представление времени в соответствии с заданным форматом. В следующем примере мы создаем объект timedelta, содержащий временной интервал в 1000 секунд, и выводим его без долей секунд:


1

2

3

4

import datetime

 

delta = datetime.timedelta(seconds=1000)

print(delta.strftime('%H:%M:%S'))

Результат выполнения кода будет следующим:


1

00:16:40

Здесь мы использовали строку формата '%H:%M:%S', чтобы вывести только часы, минуты и секунды интервала.

Единицы измерения времени, включенные в строку формата, могут варьироваться в зависимости от требуемого формата вывода. Например, чтобы вывести интервал в формате "часы:минуты", мы можем использовать следующую строку формата:


1

print(delta.strftime('%H:%M'))

Результат выполнения будет таким:


1

00:16

Также можно добавить другие единицы измерения времени в строку формата, например, дни или месяцы:


1

2

3

4

delta = datetime.timedelta(days=365, hours=6, minutes=15, seconds=30)

 

print(delta.strftime('%j day(s) %H:%M:%S'))

print(delta.strftime('%m month(s) %d day(s) %H:%M:%S'))

В результате выполнения кода мы получим:


1

2

365 day(s) 06:15:30

01 month(s) 01 day(s) 06:15:30

Перейдем к следующему способу вывода интервала времени без долей секунд — это использование метода str() объекта timedelta. Метод str() преобразует объект timedelta в строку, содержащую представление интервала времени в виде "дни, часы:минуты:секунды". Таким образом, мы можем использовать метод str() для вывода интервала без долей секунд:


1

2

3

delta = datetime.timedelta(days=365, hours=6, minutes=15, seconds=30)

 

print(str(delta).split('.')[0])

Результат выполнения кода будет следующим:


1

365 days, 6:15:30

Здесь мы использовали метод split() для разбиения строки на две части и вывода только первой части без долей секунд.

Также мы можем использовать форматирование строк для вывода интервала в нужном формате:


1

2

3

delta = datetime.timedelta(days=365, hours=6, minutes=15, seconds=30)

 

print('{0.days} days, {0.hours:02}:{0.minutes:02}:{0.seconds:02}'.format(delta))

Здесь мы использовали метод format() для форматирования строки. В фигурных скобках мы указали индекс аргумента, переданного методу format(). Мы также использовали двоеточие и формат «%02d», чтобы выровнять значения от 0 до 9.

В этой статье мы рассмотрели два способа вывода интервала времени без долей секунд с помощью модуля datetime. Метод strftime() позволяет настраивать вывод интервала в соответствии с заданным форматом, а метод str() предоставляет готовый формат вывода «дни, часы:минуты:секунды». Вы можете выбрать тот метод, который лучше всего подходит для вашего проекта.

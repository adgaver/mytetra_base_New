НАПИСАНИЕ ТЕКСТОВЫХ ФАЙЛОВ В VBA

Подход №1: Функция Write (строки в кавычках)

Запись строк данных в текстовые файлы с кавычками:

Dim fileName As String, textData As String, textRow As String, fileNo As Integer

fileName = "C:\test.txt"

fileNo = FreeFile 'Get first free file number 

textData ="Hello World!"

 

Open fileName For Output As #fileNo  'Open file for overwriting! Replace Output with Append to append

Write #fileNo, textData

Close #fileNo

Этот подход приведет к следующему результату:

"Hello, World!"

Обратите внимание на кавычки «» . Запись дополнительных строк приведет к добавлению дополнительных строк, заключенных в двойные кавычки, что может быть неудовлетворительным подходом. Посмотрите вниз на функцию печати, чтобы избежать этих кавычек.

Чтобы добавить в текстовый файл вместо перезаписи, замените For Output на For Append в Open fileName For Output As #fileNo

Подход №2: функция Print (вывод строк без кавычек)

Запись строк данных в текстовые файлы без кавычек:

Dim fileName As String, textData As String, textRow As String, fileNo As Integer

fileName = "C:\test.txt"

fileNo = FreeFile 'Get first free file number 

   

Open fileName For Output As #fileNo 'Open file for overwriting! Replace Output with Append to append

Print #fileNo, textData

Close #fileNo

Этот подход приведет к следующему результату:

1

Hello, World!

На этот раз в файле нет кавычек. В большинстве случаев вы захотите использовать функцию печати вместо функции записи .

НАПИСАНИЕ БИНАРНЫХ ФАЙЛОВ В VBA

Dim fileName As String, fileNo As Integer, testVar As Integer

fileName = "C:\test.bin"

testVar = 4

fileNo = FreeFile

Open fileName For Binary Lock Read Write As #fileNo

Put #fileNo, , testVar

Close #fileNo

С двоичными файлами вы часто будете использовать объекты, которые не имеют фиксированной длины в байтах, например, целые числа . Например, вы хотели бы сохранить строки. В таких случаях используйте тип данных объекта VBA Type .

Ниже приведен простой пример использования типа данных Type для сохранения объекта с целым числом и строкой.

Type TestType

    intVar As Integer

    strVar As String

End Type

 

Sub WriteBinary()

  Dim fileName As String, fileNo As Integer, testVar As TestType

  fileName = "C:\test.bin"

  testVar.intVar = 4

  testVar.strVar = "Hello!"

  fileNo = FreeFile

  Open fileName For Binary Lock Read Write As #fileNo

  Put #fileNo, , testVar

  Close #fileNo

End Sub

ЗАПИСЬ ФАЙЛОВ CSV В VBA

Если вы хотите сохранить рабочий лист в формате CSV, вы всегда можете обратиться к разделу «Текстовый файл» . Однако это малоэффективный подход. Если вы хотите сохранить текущий рабочий лист в виде файла, разделенного точкой с запятой ( ';' ), вы можете использовать функцию Workbook.SaveAs , как показано ниже.

PathName = "C:\test.csv"

'Assuming you want to export your current Worksheet

ActiveSheet.Move 'Move the Worksheet to a new Workbook

ActiveWorkbook.SaveAs Filename:=PathName, FileFormat:=xlCSV, CreateBackup:=False

Если вы хотите вместо этого скопировать свой рабочий лист, просто замените Move for Copy :

PathName = "C:\test.csv"

'Assuming you want to export your current Worksheet

ActiveSheet.Copy 'Copy the Worksheet to a new Workbook

ActiveWorkbook.SaveAs Filename:=PathName, FileFormat:=xlCSV, CreateBackup:=False

POWER QUERY. ФИЛЬТРАЦИЯ СПИСКОМ

У нас есть большая таблица, в которой находятся данные по продажам. Есть еще одна маленькая таблица, которая содержит только интересующие нас коды товаров. Нам нужно из большой таблицы оставить только строки с нужными кодами, которые находятся во второй таблице.

Мы решим эту задачу двумя способами:

  1. Объединение таблиц
  2. Фильтрация нужного столбца при помощи списка

Решение

Чтобы решить задачу первым способом мы воспользуемся объединением таблиц справа. Таким образом мы оставим только коды, которые находятся в таблице справа, а все остальные будут отсеяны.

Для решения задачи вторым способом мы воспользуемся функцией List.Contains, которая будет вложена в функцию Table.SelectRows.

Примененные функции

  • Excel.CurrentWorkbook
  • Table.TransformColumnTypes
  • Table.NestedJoin
  • JoinKind.RightOuter
  • Table.RemoveColumns
  • List.Contains

Код

Сначала получим данные:

let

   source = Excel.CurrentWorkbook(){[Name = "Продажи"]}[Content],

   types  = Table.TransformColumnTypes(source, {{"Дата", type date}})

in

   types

Решение с помощью объединения таблиц

Получаем вторую таблицу:

let

   source = Excel.CurrentWorkbook(){[Name = "Список"]}[Content]

in

   source

Выполняем объединение:

let

   source = Table.NestedJoin(

      data,

      {"Код"},

      table_filter,

      {"Список"},

      "Список_Таблица",

      JoinKind.RightOuter

   ),

   cols_select = Table.RemoveColumns(source, {"Список_Таблица"})

in

   cols_select

Решение с помощью List.Contains

Получаем список:

let

   Источник    = Excel.CurrentWorkbook(){[Name = "Список"]}[Content],

   list_filter = Источник[Список]

in

   list_filter

Выполняем фильтрацию:

let

   source = Table.Buffer(data),

   rows_select = Table.SelectRows(

      source,

      each List.Contains(list_filter, [Код])

   )

in

   rows_select

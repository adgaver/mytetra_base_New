to_html() : DataFrame в html


import pandas as pd

my_dict={

'NAME':['Ravi','Raju','Alex'],

'ID':[1,2,3],'MATH':[30,40,50],

'ENGLISH':[20,30,41]

}

df = pd.DataFrame(data=my_dict) # create DataFrame from dictionary

df.to_html('D:\my_html.html')



Мы сохранили данные DataFrame в виде таблицы внутри файла .html (веб-страницы).

HTML-вывод Python Pandas из DataFrame и использование таблицы-образца MySQL в качестве источника с помощью to_html()

По умолчанию у нас будет индекс как самый левый столбец. Мы можем удалить индекс, используя опцию index=False

df.to_html('D:\my_html.html',index=False)


Путь сохранения

Мы сохранили файл на диске D (корневой каталог)

df.to_html('D:\my_file.html')


Внутренний каталог данных

df.to_html('D:\data\my_file.html')


Параметры

buf: Буфер для записи.
columns: Значение по умолчанию None, записывать все столбцы. Подсписок столбцов для генерации.
col_space: Необязательно, ширина css в пикселях. Вывод <th style="min-width: 20px;">
header : Значение по умолчанию True, bool Необязательно. Печатать заголовки столбцов (True) или нет
index: Значение по умолчанию True, bool Необязательно. Печатать строку индекса (True) или нет
na_rep: Как обрабатывать записи NaN
formatters . : Функция, применяемая к элементам столбцов для форматирования.
float_format: Функция форматирования для чисел с плавающей точкой.
sparsify: Печать иерархического индекса
index_names: Bool, необязательно, по умолчанию True. Печатать имя индексов
justify: Выровнять метки столбцов. Значения left, right, center, justify, justify-all, start, end, inherit, match-parent, initial, unset.
max_rows: int, Необязательно. Максимальное количество отображаемых строк.
max_cols: int, Необязательно. Максимальное количество отображаемых столбцов.
show_dimensions: bool, по умолчанию False.
decimal: Десятичный разделитель. Пример , используется в Европе.
bold_rows: Bool, по умолчанию True, помечает метки строк жирным шрифтом.
classes: (HTML ) Класс для добавления в таблицу для стиля.
escape: Bool, по умолчанию True. Преобразует < , > и &
notebook: Bool, вывод для IPython Notebook.
border: int, ширина границы включена в тег <table>, значение по умолчанию 1, установите 0, чтобы удалить границу.
table_id: Идентификатор css для таблицы html.
render_links: По умолчанию False. Преобразует URL в ссылку
encoding:str, кодировка, по умолчанию 'utf-8'

Вывод : Строка или None в зависимости от настройки buf .

Примеры

Мы можем указать столбцы, которые будут использоватьсяindex=False

import pandas as pd

df=pd.read_excel("E:\\data\\student.xlsx") # Path of the file.

l1=['id','name','mark'] # List of columns to show in html table

df.to_html('E:\\data\\my_html.html',index=False,columns=l1)


max_rows и max_cols

df.to_html('C:\\data\\student.html',max_rows=5,max_cols=3)


удалить заголовок

df.to_html('C:\\data\\student.html',index=None,header=False)


классы

Это добавит класс в сгенерированную HTML-таблицу.

df.to_html('C:\\data\\student.html',classes="table table-striped")


col_space

Это добавит ширину строке заголовка таблицы.<th style="min-width: 50px;">id</th>

df.to_html('C:\\data\\student.html',col_space=50)


na_rep Пустые данные

Как обработать пустые данные, можно использовать na_rep='*'

import pandas as pd

import numpy as np

my_dict={

'NAME':['Ravi','Raju','Alex'],

'ID':[1,2,3],'MATH':[30,np.nan,50],

'ENGLISH':[20,30,41]

}

df = pd.DataFrame(data=my_dict) # create DataFrame from dictionary

print(df.to_html(na_rep='*'))


показать_размеры

print(df.to_html(show_dimensions=True))


Вместе с таблицей мы получим такой вывод.

<p>3 rows × 4 columns</p>


десятичная дробь

Вместо точки ( . ) в качестве десятичного разделителя можно использовать запятую, как это принято в Европе.

import pandas as pd

my_dict={

'NAME':['Ravi','Raju','Alex'],

'ID':[1,2,3],'MATH':[30,80,50],

'ENGLISH':[20.5,30.65,41.898]

}

df = pd.DataFrame(data=my_dict) # create DataFrame from dictionary

print(df.to_html(decimal=','))


table_id

Мы можем добавить один идентификатор CSS в сгенерированную таблицу.

print(df.to_html(table_id='my_table'))


Выход

<table border="1" class="dataframe" id="my_table">


Хранение части данных

Мы можем отфильтровать DataFrame и затем сохранить строки в html-файле. Для этого мы будем использовать наш файл test.csv , так как у нас больше строк.
Теперь давайте сохраним только два столбца, class и name

import pandas as pd

df=pd.read_csv('test.csv')

df=df.loc[:,['class','name']]

df = pd.DataFrame(data=df)

df.to_html('my_file.html',index=False)


Мы можем использовать различные другие фильтры для управления данными и сохранения в html-файле. Вы можете узнать больше о разделах фильтров.

Использование SQLAlchemy

Мы соберем записи из нашей таблицы образца студента в базе данных MySQL и отобразим как html с помощью to_html() .
Соберите дамп SQL таблицы образца студента ниже.
Подробнее о MySQL с подключением SQLAlchemy . Вы можете добавить путь, если хотите, чтобы файл (с тегами html) был создан с помощью (пример приведен выше)

import pandas as pd

from sqlalchemy import create_engine

my_conn = create_engine("mysql+mysqldb://userid:pw@localhost/my_db")

sql="SELECT * FROM student LIMIT 0,10 "

df = pd.read_sql(sql,my_conn)

df.to_html('D:\\df\\my_html.html')


Из файла Excel в HTML-вывод

В коде выше мы создали один DataFrame, взяв данные из таблицы базы данных MySQL. Мы можем создать DataFrame, используя любые данные Excel или любой файл CSV или из любых других источников. (проверьте здесь, чтобы создать DataFrame из 8 различных источников )
После создания DataFrame, с его помощью мы можем создать вывод HTML с помощью to_html() . Вот один пример чтения одного файла Excel в DataFrame и генерации строки, вы можете изучить другие источники, чтобы создать DataFrame и, наконец, сгенерировать теги / файл HTML.
Мы использовали read_excel() для чтения нашего образца файла student.xlsx.

df=pd.read_excel("D:\\my_data\\student.xlsx") # Path of the file.

df.to_html('D:\\my_data\\my_html.html')

df.to_html('D:\\my_data\\my_html.html')



Добавление стиля с использованием опции классов

import pandas as pd

df=pd.read_excel("E:\\data\\student.xlsx") # Path of the file.

df.to_html('E:\\data\\my_html.html',classes='center') # File created


Код выше создаст файл my_html.html по указанному пути, добавив class=center для выравнивания таблицы по центру. Верхняя часть исходного кода html находится здесь.

<table border="1" class="dataframe center">

<thead">

<tr style="text-align: right;">

-----

-----


Поскольку класс добавляется в код html, мы можем определить класс html в наших свойствах стиля . Сохраните этот код в CSS вашей страницы или внутри тега HEAD . Вот один пример.

<style>.center {

margin-left: auto;

margin-right: auto;

}</style>


Чтобы вывести HTML-вывод на консоль.

print(df.to_html(classes='center'))


Скопируйте табличные данные из URL и создайте HTML-таблицу

Выберите табличные данные с любой веб-страницы и скопируйте в буфер обмена. Затем запустите этот код, чтобы сгенерировать таблицу html для использования на любой странице html.
Используется этот пример URL. https://www.php.net/manual/en/errorfunc.constants.php

import pandas as pd

df=pd.read_clipboard(index_col='Value') # use value column as index

df=df.drop(columns='Note') # delete the Note column

print(df.to_html(classes="table table-striped"))

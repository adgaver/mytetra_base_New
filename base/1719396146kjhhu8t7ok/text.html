Могу ли я перемещаться по истории браузера с помощью Playwright?

Да, вы можете перемещаться по истории браузера с помощью Playwright. Playwright предоставляет ряд методов навигации, в том числе page.goBack()и page.goForward()которые можно использовать для навигации по истории браузера.

Вот примеры как на Python, так и на JavaScript:

Питон:

Копировать

from playwright.sync_api import sync_playwright

with sync_playwright() as p:

browser = p.chromium.launch()

page = browser.new_page()

page.goto('https://example.com')

# Now, let's navigate to another page

page.goto('https://google.com')

# You can go back to the previous page using goBack()

page.goBack()

# And you can move forward in the history using goForward()

page.goForward()

browser.close()



JavaScript:

Копировать

const playwright = require('playwright');

(async () => {

const browser = await playwright.chromium.launch();

const context = await browser.newContext();

const page = await context.newPage();

await page.goto('https://example.com');

// Now, let's navigate to another page

await page.goto('https://google.com');

// You can go back to the previous page using goBack()

await page.goBack();

// And you can move forward in the history using goForward()

await page.goForward();

await browser.close();

})();



Эти команды будут перемещаться по истории просмотров так же, как кнопки «назад» и «вперед» в браузере. Важно отметить, что page.goBack()и page.goForward()возвращают обещание, которое разрешается в ответ основного ресурса. Если навигация происходит в другом источнике, метод будет разрешаться в null.

Также имейте в виду, что если в истории нет предыдущей или следующей страницы, эти методы ничего не сделают. Вы можете проверить длину истории с помощью page.evaluate(() => window.history.length).

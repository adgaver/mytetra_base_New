Свойство Application. LanguageSettings (Excel)

  • 05.04.2019
  • Время чтения: 2 мин


Возвращает объект LanguageSettings , который содержит сведения о языковых параметрах в Microsoft Excel. Только для чтения.

Синтаксис

Expression. LanguageSettings

выражение: переменная, представляющая объект Application.

Пример

В этом примере возвращается идентификатор языка для выбранного языка при установке Microsoft Excel.

VB

Set objLangSet = Application.LanguageSettings

MsgBox objLangSet.LanguageID(msoLanguageIDInstall)

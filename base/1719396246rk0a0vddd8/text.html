Могу ли я получить доступ к исходному коду HTML страницы с помощью Playwright?

Да, вы можете получить доступ к HTML-источнику веб-страницы с помощью Playwright. Playwright — это библиотека Node.js для автоматизации браузеров Chromium, Firefox и WebKit с помощью единого API. Он позволяет вам управлять браузерами без графического интерфейса (без графического интерфейса) или без него, а также предоставляет функциональные возможности для очистки веб-страниц, моделирования взаимодействия с пользователем и многого другого.

Вот как это можно сделать как в JavaScript, так и в Python:

JavaScript

В JavaScript вы можете использовать эту content()функцию для получения HTML-содержимого страницы. Вот пример:

Копировать

const playwright = require('playwright');

(async () => {

const browser = await playwright.chromium.launch();

const context = await browser.newContext();

const page = await context.newPage();

await page.goto('https://example.com');

const htmlContent = await page.content();

console.log(htmlContent);

await browser.close();

})();



В этом сценарии мы сначала инициализируем новый контекст браузера, затем создаем новую страницу, переходим к https://example.com и затем получаем HTML-содержимое страницы, используя page.content().

Питон

Аналогично, в Python вы можете использовать эту content()функцию для получения HTML-содержимого страницы. Вот пример:

Копировать

from playwright.sync_api import sync_playwright

with sync_playwright() as p:

browser = p.chromium.launch()

context = browser.new_context()

page = context.new_page()

page.goto('https://example.com')

html_content = page.content()

print(html_content)

browser.close()



В этом сценарии мы делаем то же самое, что и в версии JavaScript: инициализируем новый контекст браузера, создаем новую страницу, переходим к https://example.com, а затем получаем HTML-содержимое страницы, используя page.content().

Помните, что эти примеры вернут HTML-содержимое страницы после выполнения любого JavaScript. Это означает, что если страница использует JavaScript для загрузки дополнительного контента, этот контент будет включен в возвращаемый HTML.

Получить текущий путь к файлу:os.path.dirname(os.path.abspath('__file__'))

Для работы с файлами и папками (функциями работы с файлами) в Python необходимо задействовать модуль os и модуль shutil.

Получите текущий рабочий каталог, то есть путь каталога, в котором работает текущий скрипт Python:os.getcwd()

Вернуть все файлы и имена каталогов в указанном каталоге:os.listdir()

Функция используется для удаления файла:os.remove()
Удалить несколько каталогов:os.removedirs（r“c：\python”）

Убедитесь, что указанный путь является файлом:os.path.isfile()

Убедитесь, что указанный путь является каталогом:os.path.isdir()

Определите, является ли это абсолютным путем:os.path.isabs()

Проверьте, существует ли данный путь действительно:os.path.exists()

Возвращает имя каталога и имя файла пути:os.path.split()

Пример:os.path.split('/home/swaroop/byte/code/poem.txt')
Результаты:('/home/swaroop/byte/code', 'poem.txt')

Отдельное расширение:os.path.splitext()

Получить имя пути:os.path.dirname()

Получить имя файла:os.path.basename()

Запустите команду оболочки:os.system()

Прочитайте и установите переменные среды:os.getenv()иos.putenv()

Дайте терминатор строки, используемый текущей платформой:os.linesep

Укажите платформу, которую вы используете:os.name

Переименовать:os.rename(old， new)

Создайте многоуровневый каталог:os.makedirs(r"c：\python\test")

Создайте один каталог:os.mkdir("test")

Получить атрибуты файла:os.stat(file)

Изменить права доступа к файлам и метку времени:os.chmod(file)

Завершить текущий процесс:os.exit()

Получить размер файла:os.path.getsize(filename)

Операции с каталогами:

  1. Создать каталог:
    os.mkdir("file")

  2. Скопируйте файл:
    shutil.copyfile("oldfile", "newfile") И oldfile, и newfile могут быть только файлами
    shutil.copy("oldfile", "newfile") oldfile может быть только папкой, newfile может быть файлом или целевым каталогом

  3. Копировать папку:
    shutil.copytree("olddir", "newdir") И olddir, и newdir могут быть только каталогами, а newdir не должен существовать

  4. Переименовать файл (каталог)
    os.rename("oldname", "newname") Файлы или каталоги используют эту команду

  5. Переместить файлы (каталоги)
    shutil.move("oldpos","newpos")

  6. Удалить файл
    os.remove("file")

  7. Удалить каталог
    os.rmdir("dir")Можно удалить только пустые каталоги
    shutil.rmtree("dir") Пустые каталоги и каталоги с контентом могут быть удалены

  8. Преобразовать каталог
    os.chdir("path") Изменить путь

Функция работы с файлом Pyhton:
os.mknod("test.txt")Создать пустой файл
fp = open("test.txt",w) Откройте файл напрямую, создайте файл, если файл не существует

Об открытом режиме:

w Открыть в режиме записи,

a Открыть в режиме добавления (начиная с EOF, создавая новые файлы при необходимости)

r + открывается в режиме чтения-записи

w + открывается в режиме чтения-записи (см. w)

+ открывается в режиме чтения-записи (см.)

рб открывается в двоичном режиме чтения

wb открывается в двоичном режиме записи (см. w)

ab открывается в двоичном режиме добавления (см. а)

rb + открывается в двоичном режиме чтения-записи (см. r +)

wb + открывается в двоичном режиме чтения-записи (см. w +)

ab + открывается в двоичном режиме чтения-записи (см. +)

fp.read([size]) #size - длина чтения в байтах

fp.readline([size]) # Читать строку, если размер определен, он может вернуть только часть строки

fp.readlines([size]) # Взять каждую строку файла в качестве члена списка и вернуть список. На самом деле его внутренний называется цикломreadline()Для достижения. Если указан параметр size, size - это общая длина прочитанного содержимого, что означает, что он может быть прочитан только для части файла.

fp.write(str) # Записать в файл str, write () не добавляет символ новой строки после str

fp.writelines(seq) # Записать все содержимое seq в файл (несколько строк пишутся одновременно). Эта функция также написана точно, без добавления чего-либо после каждой строки.

fp.close() # Закройте файл. Python автоматически закроет файл, когда он не используется, но эта функция не гарантируется, лучше всего выработать привычку к закрытию. Если файл закрыт, он сгенерирует ValueError

fp.flush() # Записать содержимое буфера на жесткий диск

fp.fileno() # Вернуть длинную "метку файла"

fp.isatty() # Является ли файл файлом конечного устройства (система Unix)

fp.tell() # Возврат к текущей позиции метки файла, с началом файла в качестве источника

fp.next() # Вернитесь к следующей строке и переместите метку файловой операции на следующую строку. Когда файл используется в выражении, например для ... в файле, он называетсяnext()Функция для достижения обхода.

fp.seek(offset[,whence]) # Переместите метку операции с файлом в положение смещения. Это смещение обычно рассчитывается относительно начала файла и обычно является положительным числом. Однако нет необходимости, если задан параметр whenence, где может быть 0 для начала с начала и 1 для начала с текущей позиции. 2 указывает, что конец файла используется в качестве источника. Обратите внимание, что если файл открывается в режиме или +, флаг операции с файлом будет автоматически возвращаться в конец файла каждый раз, когда выполняется операция записи.

fp.truncate([size]) # Обрезать файл до указанного размера, по умолчанию вырезать до позиции текущей отметки операции файла. Если размер больше размера файла, он может не изменить файл в зависимости от системы, или может использовать 0, чтобы заполнить файл до соответствующего размера, или он может быть добавлен с некоторым случайным содержимым.

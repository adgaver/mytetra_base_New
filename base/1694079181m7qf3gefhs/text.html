Объявите перечислимый тип и присвойте ему значение.

Во-первых, в VBA Enums объявляется в верхней части модуля.


Enum Grading

grExcellent = 1

grGood = 2

grNormal = 3

grMediocre = 4

grBad = 5

End Enum

Далее вы будете использовать его для объявления переменных типа и присвоения им значения, например:


Dim grStudentGrade As Grading : grStudentGrade = grGood

Enum Builder также является удобным инструментом для создания нескольких вспомогательных функций.


Перевод перечислений в их строковые значения

При проверке Enum_ToString сгенерированный код будет включать функцию, которая представляет читаемую форму для оценочного значения; он возвращает «Хорошо» вместо загадочного значения 2.


Function Grading_ToString(value As Grading) As String

Select Case value

Case grExcellent: Grading_ToString = "Excellent"

Case grGood: Grading_ToString = "Good"

Case grNormal: Grading_ToString = "Normal"

Case grMediocre: Grading_ToString = "Mediocre"

Case grBad: Grading_ToString = "Bad"

End Select

End Function

Перевод строкового значения в их значение перечисления

Когда учитель вводит значения во время оценивания, вам может потребоваться интерпретировать строковое значение как его перечисляемый аналог. Эта функция становится доступной, если вы отметите String_ToEnum:


Function Grading_ToEnum(str As String) As Grading

Select Case str

Case "grExcellent": Grading_ToEnum = grExcellent

Case "grGood": Grading_ToEnum = grGood

Case "grNormal": Grading_ToEnum = grNormal

Case "grMediocre": Grading_ToEnum = grMediocre

Case "grBad": Grading_ToEnum = grBad

End Select

End Function

Вызов этой функции может работать следующим образом (здесь с префиксом gr)


Dim grStudentGrade As Grading

Dim strGrade As String: strGrade = "Good"

grStudentGrade = Grading_ToEnum("gr" & strGrade)

Проверка того, является ли число допустимым значением Enum

Допустим, вы разрешаете учителю также вводить оценки в виде чисел. Вы можете сказать любое значение от 1 до 5, что здесь нормально, но возможно (с MS это происходит регулярно), что числа не являются четкой последовательностью.


Function Grading_IsEnumValue(lng As Long) As Boolean

Select Case lng

Case grExcellent, grGood, grNormal, grMediocre, grBad

Grading_IsEnumValue = True

End Select

End Function

Цикл по перечисляемым значениям

Чтобы обеспечить это, мы расширяем перечисление с помощью [_First] и [_Last]. Это можно использовать, среди прочего, в качестве альтернативы описанному выше Select Caseподходу в случае проверки правильности введенных данных.


Enum Grading

grExcellent = 1

grGood = 2

grNormal = 3

grMediocre = 4

grBad = 5

[_First] = 1

[_Last] = 5

End Enum

Dim i As Long, booIsValid As Boolean

Dim grStudentGrade As Grading: grStudentGrade = 4

For i = Grading.[_First] To Grading.[_Last]

If grStudentGrade = i Then

booIsValid = True

Exit For

End If

Next N

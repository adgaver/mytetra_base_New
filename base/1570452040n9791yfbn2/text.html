Как использовать функцию Наборы (Sets) в Windows 10

 Функция Наборы (Sets) - это одно из самых больших изменений в интерфейсе рабочего стола Windows 10. Почти каждое приложение теперь имеет вкладки в строке заголовка. В одном окне можно использовать вкладки из различных приложений, например таких как Проводник Windows, Microsoft Word и Microsoft Edge.


 Функция Наборы (Sets) позволяет пользователю группировать несколько приложений в виде вкладок в одном окне. Это означает, что вы можете использовать вкладки в проводнике, командной строке, блокноте и других приложениях, которые никогда не предлагали вкладки.


 Функция Наборы также позволяет открыть вкладку браузера Microsoft Edge практически в любом окне. Например, вы можете работать над документом в Microsoft Word, открыть вкладку браузера Edge, чтобы найти что-то в интернете, а затем переключиться обратно в документ, и все это можно сделать из панели вкладок, без переключения окон.

 С помощью функции Наборы (Sets ) можно объединять в одном окне несколько приложений для решения различных задач. Принцип работы похож на браузер, где постоянно открыто несколько вкладок.

 Функция Наборы (Sets) доступна для участников программы предварительной оценки Windows, в сборках Redstone 5 Insider Preview.


Как использовать Наборы (Sets)

 Наборы легко обнаружить. Большинство настольных приложений теперь имеют вкладки в строке заголовка. Нажмите кнопку Создать вкладку на панели, чтобы открыть новую вкладку.

 После нажатия кнопки Создать вкладку откроется новая вкладка с вкладкой браузера Microsoft Edge. Вы можете просматривать веб-страницы здесь, так же, как если бы вы просматривали их в браузере Edge.

 Также есть возможность открывать приложения с этой новой вкладки. Для этого выберите Приложения

 Далее прокрутите вниз скроллбар и откройте нужное приложение, например приложение OneNote.

 Напомню, что не все приложения из этого списка открываются в новой вкладке, если приложение не поддерживает наборы, то оно откроется в новом окне.


 Вы можете самостоятельно собрать наборы для себя. Когда вы откроете несколько разных окон приложений, вы увидите панель вкладок в каждом из них. Чтобы объединить эти приложения в одном окне, перетащите вкладку из панели вкладок одного приложения в панель вкладок другого приложения. Повторите этот процесс, чтобы добавить дополнительные вкладки приложений в одно окно.

 Вы можете легко открыть новую вкладку используя например Проводник Windows. В проводнике можно нажать сочетание клавиш Ctrl+T, при этом откроется новая, точно такая же вкладка.


 Можно также щелкнуть правой кнопкой мыши на нужной папке в проводнике и выбрать в открывшемся контекстном меню пункт Открыть в новой вкладке

 Как и в веб-браузере, вы можете нажать правой кнопкой мыши на вкладке, чтобы найти возможности управления ими. Щелкните правой кнопкой мыши вкладку и в открывшемся контекстном меню выберите нужное действие: Закрыть вкладку, Закрыть другие вкладки, Закрыть вкладки справа, Переместить в новое окно

 Вы также можете перетаскивать вкладки, чтобы изменить их порядок.

 Функция Наборы (Sets), также запоминает предыдущие вкладки, связанные с приложением. Вы можете просмотреть их, нажав кнопку Предыдущие вкладки слева от вкладок.

 Например, предположим, что вы ранее работали над документом в блокноте, и у вас была вкладка браузера Edge, открытая для определенного веб-сайта, но с тех пор вы закрыли окно. В следующий раз, когда вы откроете блокнот, вы можете нажать кнопку Предыдущие вкладки в левом верхнем углу окна, чтобы найти и открыть эту вкладку браузера.

 В этом списке также отображаются вкладки приложений, которые Вы использовали с текущим приложением,что позволяет быстро открывать наборы приложений.



Сочетания клавиш для наборов

 Стоит также отметить то, что разработчики добавили новые сочетания клавиш для более удобной работы с наборами вкладок.

Ctrl + Win + Tab переключение на следующую вкладку

Ctrl + Win + Shift + Tab переключение на предыдущую вкладку

Ctrl + Win + T открыть новую вкладку

Ctrl + Win + W закрыть текущую вкладку

Ctrl + Win + Shift + T открыть последнюю закрытую вкладку

Ctrl + Windows + 1-9: переключение на определенную вкладку на панели вкладок.
 Например, Ctrl + Win + 1 переключается на первую вкладку, в то время как Ctrl + Win + 4 переключается на четвертую вкладку.

Как настроить Наборы (Sets)

 Чтобы найти настройки этой функции, последовательно откройте:

Параметры Windows ➯ Система ➯ Многозадачность


 Прокрутите боковой скроллбар вниз к разделу Наборы

 Параметр Приложения и веб-сайты автоматически открываются в новом управляет отображением приложений и веб-сайтов, которые вы открываете. По умолчанию он установлен в Tab, но вы можете выбрать Окно, тогда приложения будут открываться в новом окне.

 Опция При нажатии клавиш ALT + TAB отображаются недавно использованные элементы позволяет управлять работой переключателя Alt+Tab. По умолчанию Alt + Tab теперь показывает как вкладки, так и окна, что позволяет легко переключаться между вкладками. Если вы предпочитаете старое поведение, вы можете выбрать Только окна вместо Окна и вкладки

 Существует также опция Приложения, добавленные в этот список, не могут быть включены в наборы вкладок. Это позволяет отключить наборы для определенного приложения. Чтобы отключить вкладки наборов для приложения, нажмите Добавить приложение и выберите приложение из списка.

 Например, если добавить в этот список Проводник Windows, то вкладка «наборы» исчезнет из строки заголовка проводника, и вы больше не сможете создавать вкладки проводника. Это будет полезно, если Вы не хотите использовать вкладки наборов для определенного приложения или если наборы вызывают проблемы с приложением.

 Функция Наборы (Sets) очень полезный инструмент для быстрого выполнения различных задач. В дальнейшем, в процессе разработки функция Наборы будет изменяться, но уже и в данный момент она работает не плохо.

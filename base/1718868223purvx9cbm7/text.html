Как я могу извлечь путь к папке из пути к файлу в Python?


>>> import os

>>> '\\'.join(existGDBPath.split('\\')[0:-1])

'T:\\Data\\DBDesign'



Хотя я бы рекомендовал использовать для этого функцию os.path.dirname, вам просто нужно передать строку, и она сделает всю работу за вас. Поскольку вы, похоже, работаете в Windows, рассмотрите возможность использования abspath функции тоже. Пример:

>>> import os

>>> os.path.dirname(os.path.abspath(existGDBPath))

'T:\\Data\\DBDesign'



Если вы хотите получить и имя файла, и путь к каталогу после разделения, вы можете использовать os.path.split функцию, которая возвращает кортеж следующим образом.

>>> import os

>>> os.path.split(os.path.abspath(existGDBPath))

('T:\\Data\\DBDesign', 'DBDesign_93_v141b.mdb')

Вот небольшая коллекция скриптов, которые я создал, скачал, изменил и т. д. для сжатия, редактирования, добавления, ремультиплексирования видеофайлов, а также для преобразования файлов MKV без необходимости повторного рендеринга.

Скопируйте код и сохраните его как файл .bat, сохраните его в папке bin ffmpeg или в любой командной строке, если ffmpeg был установлен с указанием пути к среде.

Чтобы использовать сценарии, просто перетащите видеофайл в созданный вами файл .bat.

Загрузите ffmpeg отсюда: https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-full.7z.

Если FFMPEG отсутствует или путь не настроен, выполните следующие действия.

Загрузите последний «статический» пакет ffmpeg .

Просмотрите архив до binподпапки, содержащей исполняемые файлы ffmpeg, ffprobe и ffplay. Разархивируйте binпапку (в этом примере мы будем использовать )C:\ffmpeg\bin

Зарегистрируйте ffmpeg, ffprobe и ffplay в переменных среды.

Через командные строки:
нажмите клавишу Windows, запишите cmd, нажмите Enterи введите следующую команду, строку за строкой:

set ffmpeg=C:\ffmpeg\bin
set ffprobe=C:\ffmpeg\bin
set ffplay=C:\ffmpeg\bin

Через пользовательский интерфейс:
Перейдите к:

My Computer/Environment variables
Add Name: ffmpeg Value: C:\ffmpeg\bin
Add Name: ffprobe Value: C:\ffmpeg\bin
Add Name: ffplay Value: C:\ffmpeg\bin

Звук MKV 5.1 в стерео без рендеринга (полезно, если исходный звук не воспроизводится на телевизоре)

Сведите звук 5.1 в стерео для файлов MKV со звуком 5.1, который невозможно воспроизвести.

@echo off


cd /D %~p0



echo Convert Audio to Stereo ...

for %%i in (%*) do if not exist "%~p1%%~ni_Stereo.mkv" (

ffmpeg.exe -i "%%~i" -map 0 -c copy -c:a aac -ac 2 "%~p1%%~ni_Stereo.mkv"

)

popd

echo Done..


Извлеките аудиодорожку № 2 из видео MKV и экспортируйте ее как аудиофайл mp4.

Этот скрипт извлекает аудиодорожку №2 из перетаскиваемого MKV. Если вам нужно извлечь дорожку №1, измените «-map 0:a:1» на «-map 0:a:0» (вместо этого дорожка начинается с 0). из 1)

@echo off


cd /D %~p0



echo Extracting audio track 2 ...

for %%i in (%*) do if not exist "%~p1%%~ni_Audio.mp4" (

ffmpeg -i "%%~i" -map 0:a:1 -c copy -c:a aac -ac 2 "%~p1%%~ni_Audio.mp4"

)

popd

echo Done..

Конвертировать OGV в MP4

Конвертируйте любой файл OGV в MP4 с помощью быстрого преобразования и использования h264, просто перетащите файл в сценарий.

@echo off

setlocal enabledelayedexpansion


REM Check if FFmpeg is installed

where ffmpeg >nul 2>nul

if %errorlevel% neq 0 (

echo FFmpeg is not installed or not in the system PATH. Please install FFmpeg and try again.

pause

exit /b 1

)


REM Check if a file is provided

if "%~1" == "" (

echo Please drag and drop an OGV file onto this script.

pause

exit /b 1

)


REM Input and output file paths with quotes

set "inputFile=%~1"

set "outputFile=%~dpn1.mp4"


REM Run FFmpeg command with quotes around file paths

ffmpeg -i "!inputFile!" -c:v libx264 -crf 22 -c:a aac -b:a 128k -strict -2 "!outputFile!"


REM Check FFmpeg exit code

if !errorlevel! neq 0 (

echo An error occurred during conversion.

pause

exit /b 1

)


echo Conversion successful. Output file: "!outputFile!"

pause

ПОСЛЕДОВАТЕЛЬНОСТЬ ВИДЕО В JPG

Конвертируйте любое видео в формат jpg.

@echo off

setlocal EnableExtensions DisableDelayedExpansion


REM Check if the batch file is called with at least one non-empty argument string.

if "%~1" == "" echo INFO: Drag an MP4 file onto this script to begin frame extraction.& pause & exit /B


set "inputFile=%~1"


REM Create the subdirectory input in the directory named after the video file.

md "%~dpn1\input" 2>nul

if not exist "%~dpn1\input\" echo ERROR: Could not create the directory: "%~dpn1\input"& pause & exit /B 2


REM Extract frames from the MP4 file.

ffmpeg.exe -i "%~1" -qscale:v 1 -qmin 1 -qmax 1 -vsync 0 "%~dpn1\input\image%%03d.jpg"

if errorlevel 1 rd "%~dpn1\input" 2>nul & echo ERROR: Failed to extract the frames from: "%~1"& pause & exit /B 1


echo Extraction complete.

pause

endlocal

REM Check FFmpeg exit code

КОНВЕРТИРОВАТЬ ПОСЛЕДОВАТЕЛЬНОСТЬ IMG В MP4

@echo off

REM Input and output file paths with quotes

:: The script expects the first parameter to be the image file.

set "imageFile=%~1"

:: Extract the filename without extension for the output video name.

for %%A in ("%imageFile%") do set "filename=%%~nA"


set "inputFile=%filename%.jpg"

set "outputFile=%filename%.mp4"



ffmpeg ^

-framerate 24 ^

-start_number 1 ^

-i "TURN_%%04d.jpg" ^

-c:v libx264 -pix_fmt yuv420p -crf 17 ^

-preset faster -tune fastdecode ^

"TURNTABLE.mp4"

if NOT ["%errorlevel%"]==["0"] goto:error

echo [92m%~n1 Done![0m


shift

if "%~1" == "" goto:end

goto:again


:error



echo [93mThere was an error. Please check your input file or report an issue on github.com/L0Lock/FFmpeg-bat-collection/issues.[0m

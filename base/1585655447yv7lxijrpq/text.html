Connection String Home Page

Welcome to the world's largest collection of connection strings on the Internet! 

On this web page, you'll find links to sample ADO and ADO.NET connection strings for .NET Data Providers,  OLE DB Providers, and ODBC Drivers (DSN/DSN-Less), from Microsoft, Oracle, and many other companies.

These sample connection strings are compiled by Carl Prothman.

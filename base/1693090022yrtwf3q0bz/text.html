setx PATH "%path%;yourpath"

редактировать и ps:

Вы можете определить букву диска без ввода, если знаете метку диска, что-то вроде этого:

@echo off
set label=DRIVENAME
set cmd=WMIC logicaldisk WHERE volumename^^="%label%" GET caption


FOR /F "tokens=1" %%G IN ('%cmd% ^| find ":"')DO set pydrive=%%G

echo %pydrive%\pathtopython
rem do your stuff here

простаивание, запущенное внутри пакета, наследует путь, но другие экземпляры не будут. Трудно проверить окончательно жестко.


Вы можете вызвать Python с помощью этой командной строки:

python -c"import idlelib.PyShell;idlelib.PyShell.main()"

и он будет вызывать оболочку IDLE без необходимости вводить что-либо.

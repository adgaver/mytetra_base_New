Получить список элементов в playwright Python

Опубликовано пользователем arilio666Ср, 10.05.2023 - 13:57

В драматургическом Python мы можем получить количество списков элементов на странице или даже получить утверждения из списка элементов на странице.

На этой странице есть несколько заголовков, всего их 10. Мы получим список тегов и будем ими управлять.

1. Получите количество присутствующих элементов.

from playwright.sync_api import sync_playwright, expect

with sync_playwright() as p:

browser = p.chromium.launch(headless=False)

page = browser.new_page()

page.goto('https://www.programsbuzz.com/search/node?keys=playwright+java')

listOfElements = page.locator("//h3[@class='search-result__title']")

expect(listOfElements).to_have_count(10)


В списке десять пунктов. Используя утверждение подсчета, мы можем это проверить.

2. N-й контент-фильтр

Используя n-й метод, мы можем изолировать текстовое содержимое списка элементов и распечатать его.

from playwright.sync_api import sync_playwright, expect

with sync_playwright() as p:

browser = p.chromium.launch(headless=False)

page = browser.new_page()

page.goto('https://www.programsbuzz.com/search/node?keys=playwright+java')

listOfElements = page.locator("//h3[@class='search-result__title']")

eleText = listOfElements.nth(1).text_content()

print(eleText)


3. Отображение списка текстов

Мы также можем использовать метод allTextContents и распечатать все тексты из списка элементов.

from playwright.sync_api import sync_playwright, expect

with sync_playwright() as p:

browser = p.chromium.launch(headless=False)

page = browser.new_page()

page.goto('https://www.programsbuzz.com/search/node?keys=playwright+java')

listOfElements = page.locator("//h3[@class='search-result__title']")

eleText = listOfElements.all_text_contents()

print(eleText)

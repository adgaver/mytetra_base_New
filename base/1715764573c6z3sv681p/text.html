Преобразование строки в DateTime в Python

Последнее обновление : 20 декабря 2023 г.

Улучшить

Работа с датами и временем - обычная задача в программировании, и Python предлагает мощный модуль datetime для обработки операций, связанных с датой и временем.

В этой статье мы собираемся преобразовать строку формата ‘гггг-мм-дд’ (гггг-мм-дд означает год-месяц-день) в объект DateTime с помощью Python.


Пример:

Input: '2023-07-25'

Output: 2023-07-25 11:30:00

Explanation: In This, we are converting the string 'yyyy-mm-dd' to Datetime format in Python.


Примечание: Существует несколько способов записи дат в строковом формате, таком как гггг-мм-дд, дд / мм / гггг и т.д. Мы можем изменить любой строковый формат даты на формат DateTime, используя эти методы.

Как преобразовать строки в DateTime в Python?

Мы можем преобразовать строку в DateTime, используя различные подходы и методы. Выберите подходящий метод, исходя из ваших требований, и корректно обрабатывайте недопустимые строки даты, чтобы обеспечить выполнение ваших программ на Python . Давайте рассмотрим каждый метод ниже:

  1. Использование strptime()
  2. С помощью dateutil.parser.parse()
  3. Используя datetime.strptime()

Преобразуйте строку Date в DateTime с помощью strptime()

Мы можем преобразовать строку date в объект DataTime, используя функцию strptime() из модуля datetime с помощью Input(). Мы будем использовать формат ‘%Y /% m /% d’, чтобы преобразовать строку в DateTime.

Пример:

Python3


import datetime

 

# datetime in string format for may 25 1999

input = '2021/05/25'

format = '%Y/%m/%d'

 

# convert from string format to datetime format

datetime = datetime.datetime.strptime(input, format)

 

# get the date from the datetime using date()

# function

print(datetime.date())

Вывод:

2021-05-25



Преобразуйте строку в объекты DateTime в Python с помощью dateutil.parser.parse()

Мы можем преобразовать строку date в формат DateTime, используя функцию dateutil.parser.parse() из библиотеки dateutil. Это обеспечивает более гибкий подход к разбору и преобразованию строк даты.

Python3


from dateutil.parser import parse

 

def convert_to_datetime(input_str, parserinfo=None):

    return parse(input_str, parserinfo=parserinfo)

 

# Example usage

date_string = '2023-07-25'

result_datetime = convert_to_datetime(date_string)

print(result_datetime)

Вывод:

2023-07-25 00:00:00

Преобразуйте список строковых дат в DateTime с помощью datetime.strptime()

Мы можем преобразовать список строковых дат в объект DateTime с помощью datetime.strptime(). Метод datetime.strptime() в Python позволяет нам проанализировать строку даты в указанном формате и преобразовать ее в объект datetime.

Python3


# import the datetime module

import datetime

 

# datetime in string format for list of dates

input = ['2021/05/25', '2020/05/25', '2019/02/15', '1999/02/4']

 

# format

format = '%Y/%m/%d'

for i in input:

   

    # convert from string format to datetime format

    # and get the date

    print(datetime.datetime.strptime(i, format).date())

Вывод:

2021-05-25

2020-05-25

2019-02-15

1999-02-04



Устранение ошибок strptime()

strptime() выдаст вам ValueError, если входная строка не может быть проанализирована в заданном формате.

Вы можете попробовать блоки ‘try и except’, чтобы найти возможное решение ValueError.

Пример:

Python3


from datetime import datetime

# error as 2021 is not a leap year

date_string = "2021-02-29"

date_format = "%Y-%m-%d"

# Attempt to convert string to datetime

try:

    date_object = datetime.strptime(date_string, date_format)

    print(date_object)

except ValueError as e:

    print("Error:", e)

Вывод

Error: day is out of range for month




Мы рассмотрели различные методы преобразования строки даты в формат DateTime в Python. Формат DateTime в Python очень полезен при работе с данными в виде дат, например, при нахождении количества дней между двумя датами.

Очень важно сохранять даты в формате DateTime, если вам нужно выполнять операции с датой. Интерпретатор будет обрабатывать “гггг-мм-дд” как строку, если вы ее не измените, что сделает невозможным выполнение операции DateTime над ней.

Включение вкладок Windows 10

Чтобы включить вкладки в Windows 10 с помощью реестра, выполните следующие действия:

  1. Комбинацией клавиш Win + R откройте окно «Выполнить».
  2. В окне «Выполнить» введите «regedit» и нажмите «Enter».
  3. В редакторе реестра перейдите к следующему ключу:


    4. HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\CloudStore\Store\Cache\DefaultAccount\$$windows.data.tabshell\Current

  4. Справа установите значение BINARY с именем Data в следующее значение:

    5. 02,00,00,00,51,1a, 48, fb, c1, cd, d3,01,00,00,00,00,43,42 , 01,00,00

  5. Чтобы изменения, внесенные с помощью настройки реестра, вступили в силу, вам нужно выйти и войти в свою учетную запись пользователя.

Чтобы сэкономить ваше время, вы можете загрузить готовые к использованию файлы реестра ниже.

Отключение вкладок Windows 10

Чтобы отключить множество вкладок в Windows 10 с помощью настройки реестра, выполните следующие действия.

  1. Комбинацией клавиш Win + R откройте окно «Выполнить».
  2. В окне «Выполнить» введите «regedit» и нажмите «Enter».
  3. В редакторе реестра перейдите к следующему ключу:


    4. HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\CloudStore\Store\Cache\DefaultAccount\$$windows.data.tabshell\Current

  4. Справа установите значение BINARY с именем Data в следующее значение:

    5. 02,00,00,00,7a, fa, 5c, b5, bd, cd, d3,01,00,00,00,00,43,42 , 01,00,02,00,00

  5. Чтобы изменения, внесенные с помощью настройки реестра, вступили в силу, вам нужно выйти и войти в свою учетную запись пользователя.
